TOPIC 10

확률은 AI랑 무슨 관계야?

Q. AI가 확률로 예측한다는 게 뭐야?

Part 1. AI는 왜 확률로 예측해?

AI가 "85% 고양이"라고 하잖아. 왜 확률로 말해?

AI는 100% 확신하지 못해. 인간처럼.

확신의 정도: 사람: "이거 고양이 같은데... 85% 확신함" AI: P(고양이) = 0.85 → 불확실성을 확률로 표현 좋은 점: 1. 얼마나 확신하는지 알 수 있음 - 99% → 거의 확실 - 51% → 애매함 2. 여러 선택지의 가능성 비교 고양이: 70% 개: 25% 토끼: 5% 3. 위험 관리 가능 의료 진단에서 60%만 확신? → 추가 검사 필요

AI는 본질적으로 확률적 모델

Part 2. 확률 기초 복습

확률이 뭐였지?

간단히 복습해볼게:

확률 (Probability): P(사건) = 원하는 경우의 수 / 전체 경우의 수 주사위 예시: P(6 나옴) = 1/6 ≈ 0.167 동전 예시: P(앞면) = 1/2 = 0.5 규칙: 1. 0 ≤ P(A) ≤ 1 → 확률은 0~1 사이 2. P(전체) = 1 → 모든 경우의 합 = 1 3. P(A or B) = P(A) + P(B) - P(A and B) → 합의 법칙 4. P(A and B) = P(A) × P(B|A) → 곱의 법칙

AI도 이 규칙을 따라.

Part 3. 조건부 확률

조건부 확률은 뭐야?

P(A|B) = "B가 일어났을 때 A가 일어날 확률"

AI 학습의 핵심이야.

예시: P(비|구름) = "구름 낀 날 비가 올 확률" = 0.4 (40%) P(비) = "그냥 비가 올 확률" = 0.1 (10%) → 구름이라는 조건이 있으면 확률이 달라짐! AI에서: P(고양이|이미지) = "이 이미지가 주어졌을 때 고양이일 확률" → 이미지 특징을 보고 확률 계산 → 이게 AI의 예측!

베이즈 정리 (Bayes' Theorem):

P(A|B) = P(B|A) × P(A) / P(B)

→ 원인과 결과를 뒤집을 수 있음

Part 4. 확률 분포는 뭐야?

확률 분포(Probability Distribution)는?

모든 가능한 결과와 그 확률을 나타낸 것

이산 확률 분포: 주사위: 1: 1/6 2: 1/6 3: 1/6 4: 1/6 5: 1/6 6: 1/6 → 합 = 1 분류 문제: 고양이: 0.7 개: 0.2 토끼: 0.1 → 합 = 1 (필수!) 연속 확률 분포: 정규분포, 균등분포 등 → 다음 토픽에서 자세히

AI의 출력 = 확률 분포

Softmax 함수로 만들어. (13번 토픽에서 배울 거야)

Part 5. AI는 어떻게 확률을 계산해?

구체적으로 어떻게 확률을 만드는 거야?

과정을 볼게:

이미지 분류 예시: 1. 이미지 입력 [픽셀 데이터] 2. 뉴럴 네트워크 통과 → 여러 층 거침 → 마지막 층 출력 (로짓): [2.1, 0.5, -0.3] 3. Softmax 적용 e^2.1 / (e^2.1 + e^0.5 + e^-0.3) = 0.72 e^0.5 / (e^2.1 + e^0.5 + e^-0.3) = 0.15 e^-0.3 / (e^2.1 + e^0.5 + e^-0.3) = 0.07 결과: [0.72, 0.15, 0.07] (합=0.94... 반올림 오차) 4. 확률 해석 고양이: 72% 개: 15% 토끼: 7% 5. 예측 가장 높은 확률 → "고양이"

핵심:

1. 뉴럴 네트워크가 점수 계산

2. Softmax로 확률로 변환

3. 가장 높은 확률 선택

Part 6. ChatGPT는 왜 매번 다른 답을 해?

같은 질문에 왜 대답이 달라져?

확률적 샘플링을 쓰기 때문이야.

단어 예측: "나는 ___ 좋아해" 확률 분포: "사과를": 40% "음악을": 30% "여행을": 20% "책을": 10% 결정적 방법: 항상 가장 높은 거 선택 → "사과를" → 매번 같은 답 확률적 방법: 확률에 따라 랜덤 선택 → 40% 확률로 "사과를" → 30% 확률로 "음악을" → 매번 다를 수 있음! Temperature 파라미터: - 낮음 (0.1): 거의 결정적 - 중간 (0.7): 균형 - 높음 (1.5): 매우 창의적 (랜덤)

이 덕분에 AI가 창의적인 답을 만들 수 있어.

핵심 정리

  • AI는 확률로 예측 - 불확실성을 수치화
  • 조건부 확률이 핵심 - P(출력|입력)
  • 확률 분포 생성 - 모든 선택지의 확률
  • 합은 항상 1 - 확률의 기본 규칙
  • Softmax로 변환 - 점수를 확률로
  • 확률적 샘플링 - 창의적 출력 가능
