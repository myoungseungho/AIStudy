TOPIC 01

AI에 수학이 왜 필요해?

Q. 코딩만 하면 안 돼?

Part 1. 수학 없이 AI 만들 수 없어?

파이썬 배우면 AI 만들 수 있다고 했잖아. 근데 왜 수학이 나와?

좋은 질문이야. 파이썬은 도구고, 수학은 원리야.

비유하자면:

비유: 자동차 운전 vs 자동차 설계 - 운전만 할 거면 → 핸들/브레이크만 알면 됨 - 자동차 만들 거면 → 물리학/역학 알아야 함 AI도 마찬가지: - 만들어진 AI 쓸 거면 → 코드만 알면 됨 - AI를 이해하고 개선할 거면 → 수학 알아야 함
그럼 코딩만 할 거면 수학 안 배워도 돼?

이론적으로는 그래.

하지만 문제는 이거야:

AI가 왜 이상한 결과를 내는지, 어떻게 고쳐야 하는지 모르면 그냥 블랙박스를 쓰는 거야.

Part 2. 수학이 어디에 쓰이는데?

구체적으로 수학이 어디에 쓰이는 거야?

AI의 모든 단계에 수학이 들어가.

단계 쓰이는 수학 왜 필요한가
데이터 표현 벡터, 행렬 이미지/텍스트를 숫자로 변환
학습 미분, 경사하강법 모델을 개선하는 방법 계산
예측 확률, 통계 결과를 확률로 표현
평가 손실함수, 로그 모델의 정확도 측정

보이지? 수학 없이는 AI가 작동조차 안 해.

Part 3. 실제로 어떻게 쓰여?

예를 들어서 설명해줄래?

간단한 예시: 고양이 사진 인식 AI

1단계: 데이터 표현 (벡터/행렬) 사진 → 픽셀 → 숫자 배열 [ [255, 200, 100, ...], [180, 220, 150, ...], ... ] → 이미지를 숫자로 바꿈 2단계: 학습 (미분/경사하강법) 현재 예측: "개" (틀림) 정답: "고양이" → 오차 계산 → 미분으로 어느 방향으로 수정할지 계산 → 경사하강법으로 조금씩 개선 3단계: 예측 (확률) 새 사진 입력 → 고양이: 85% 개: 10% 토끼: 5% → 확률이 가장 높은 "고양이" 선택

이 과정 전체가 수학 계산의 연속이야.

코드는 이 수학을 실행하는 도구일 뿐이야.

Part 4. 수학을 얼마나 알아야 해?

그럼 대학 수학을 다 알아야 해?

아니. 생각보다 많지 않아.

AI에 필요한 수학 (우선순위 순): 1. 선형대수 (Linear Algebra) - 벡터, 행렬, 곱셈 → AI의 핵심 언어 2. 미적분 (Calculus) - 미분, 편미분, 체인룰 → 학습의 핵심 원리 3. 확률/통계 (Probability & Statistics) - 확률, 분포, 평균, 분산 → 예측과 평가 4. 기초 수학 - 로그, 지수, 시그마 표기법 → 수식 읽기
이거 배우는 데 얼마나 걸려?

AI 수준으로만 배우면 1-2개월이면 충분해.

수학과 학생처럼 증명까지 다 알 필요는 없어.

"이게 뭔지, 어디에 쓰이는지"만 알면 돼.

Part 5. 수학 vs 코딩 비중은?

그럼 수학이 더 중요해? 코딩이 더 중요해?

둘 다 필요해. 하지만 역할이 달라.

구분 수학 코딩
역할 원리 이해 구현 도구
언제 쓰나 설계/디버깅/개선 실제 구현/실행
없으면 블랙박스만 쓸 수 있음 아무것도 못 만듦
학습 방법 개념 이해 중심 실습 중심

현실적으로:

초반 → 수학 개념 이해 (30%)

중반 → 코딩으로 구현 연습 (70%)

고급 → 논문 읽고 새 모델 설계 (수학 50%, 코딩 50%)

Part 6. 수학 못하는데 AI 포기해야 해?

나 수학 진짜 못하는데... AI는 나랑 안 맞는 건가?

절대 아니야.

AI 수학은 학교 수학이랑 완전히 달라.

학교 수학 vs AI 수학 학교 수학: - 복잡한 증명 - 정확한 풀이 과정 - 손으로 계산 → 너무 어려움 AI 수학: - 개념만 이해 - 컴퓨터가 계산함 - 코드로 바로 확인 가능 → 훨씬 쉬움 예시: 학교: "이차방정식 근의 공식 증명하시오" AI: "미분이 뭔지 알고, 코드로 돌려보면 됨"

핵심은 이거야:

수학을 "계산 도구"가 아니라 "사고 도구"로 배우는 거야.

"이게 왜 필요한지, 어떻게 쓰이는지"만 알면 돼.

핵심 정리

  • 수학은 AI의 언어 - 데이터 표현, 학습, 예측 모두 수학으로 이루어짐
  • 코딩은 도구, 수학은 원리 - 둘 다 필요하지만 역할이 다름
  • 많이 필요하지 않음 - 선형대수, 미적분, 확률/통계 정도면 충분
  • AI 수학은 쉬움 - 증명보다는 개념 이해와 활용이 중요
  • 1-2개월이면 충분 - AI 수준의 수학은 생각보다 빨리 배울 수 있음
