TOPIC 02

벡터가 뭐야?

Q. 숫자 여러 개를 묶어놓은 거?

Part 1. 벡터가 뭔데?

벡터가 뭐야? 숫자 여러 개 묶어놓은 거 아니야?

맞아. 그게 정확한 이해야.

벡터는 "순서가 있는 숫자들의 집합"이야.

벡터 예시: v = [3, 5, 2] → 3개의 숫자를 순서대로 묶어놓은 것 → 첫 번째 값: 3 → 두 번째 값: 5 → 세 번째 값: 2 다른 예시: a = [1.5, 2.3, 0.8, 4.1] ← 4개짜리 벡터 b = [10, 20] ← 2개짜리 벡터 c = [0, 0, 0, 0, 0] ← 5개짜리 영벡터
그럼 그냥 리스트랑 똑같은 거네?

프로그래밍 관점에서는 비슷해.

하지만 수학적 의미가 달라.

구분 리스트 벡터
의미 그냥 데이터 묶음 공간 속 점 또는 방향
연산 append, remove 등 덧셈, 곱셈, 내적 등
AI에서 데이터 저장 특징 표현

Part 2. AI에서 왜 벡터를 쓰는데?

숫자 하나하나 따로 쓰면 안 돼? 왜 벡터로 묶어?

여러 특징을 한 번에 표현하려고.

예를 들어볼게:

사과 특징 표현: 개별 숫자로: 색깔 = 0.8 (빨강 정도) 크기 = 0.6 (중간 크기) 당도 = 0.9 (매우 달음) → 따로따로 관리해야 함 벡터로: 사과 = [0.8, 0.6, 0.9] → 한 번에 관리 가능 → 계산도 한 번에! 배 = [0.3, 0.7, 0.7] 포도 = [0.5, 0.2, 0.8] → 과일들을 쉽게 비교할 수 있음

보이지? 벡터로 표현하면 여러 특징을 하나의 단위로 다룰 수 있어.

Part 3. 벡터로 뭘 할 수 있어?

벡터로 어떤 계산을 하는데?

주로 이 3가지를 써:

1. 벡터 덧셈 a = [1, 2, 3] b = [4, 5, 6] a + b = [1+4, 2+5, 3+6] = [5, 7, 9] → 같은 위치끼리 더함 2. 스칼라 곱 (숫자 곱하기) a = [1, 2, 3] 2 * a = [2*1, 2*2, 2*3] = [2, 4, 6] → 모든 원소에 같은 숫자를 곱함 3. 내적 (Dot Product) ⭐ 가장 중요 a = [1, 2, 3] b = [4, 5, 6] a · b = 1*4 + 2*5 + 3*6 = 4 + 10 + 18 = 32 → 같은 위치끼리 곱한 후 전부 더함 → 결과는 숫자 하나!
내적은 왜 중요한 거야?

내적은 "유사도"를 측정하는 핵심 도구야.

내적으로 유사도 측정: 사과 = [0.8, 0.6, 0.9] 토마토 = [0.9, 0.5, 0.3] 사과 · 토마토 = 0.8*0.9 + 0.6*0.5 + 0.9*0.3 = 0.72 + 0.3 + 0.27 = 1.29 사과 = [0.8, 0.6, 0.9] 배 = [0.3, 0.7, 0.7] 사과 · 배 = 0.8*0.3 + 0.6*0.7 + 0.9*0.7 = 0.24 + 0.42 + 0.63 = 1.29 → 내적 값이 클수록 비슷함 → AI가 "얼마나 비슷한지" 판단할 때 사용

Part 4. 실제로 어떻게 쓰여?

실제 AI에서는 어떻게 쓰이는 건데?

모든 게 벡터로 표현돼.

이미지 벡터 변환: 28x28 흑백 이미지 (MNIST 숫자) → 28 x 28 = 784개의 픽셀 → 각 픽셀 값을 일렬로 늘어놓음 [0.2, 0.8, 0.9, 0.1, ..., 0.3] ← 784차원 벡터 → 이 벡터가 "숫자 3"을 나타냄 단어 벡터 변환 (Word Embedding): "사과" = [0.2, 0.8, 0.1, ..., 0.5] ← 300차원 벡터 "배" = [0.3, 0.7, 0.2, ..., 0.4] "자동차" = [0.9, 0.1, 0.8, ..., 0.2] → 단어를 숫자로 표현 → 비슷한 단어는 벡터도 비슷함

이해됐어?

AI는 모든 데이터를 벡터로 바꿔서 처리해.

이미지도, 텍스트도, 음성도 전부 벡터야.

Part 5. 벡터의 길이는 뭐야?

벡터의 "크기"나 "길이"는 뭘 말하는 거야?

두 가지 의미가 있어:

1. 차원 (Dimension) = 벡터에 숫자가 몇 개 들어있나 v = [1, 2, 3] → 3차원 벡터 w = [5, 8, 2, 1, 9] → 5차원 벡터 2. 크기/노름 (Magnitude/Norm) = 원점에서 얼마나 떨어져 있나 v = [3, 4] |v| = √(3² + 4²) = √(9 + 16) = √25 = 5 → 피타고라스 정리 사용 → 벡터가 얼마나 "강한지" 나타냄
AI에서는 어디에 쓰여?

정규화(Normalization)에 많이 써.

벡터 정규화: v = [3, 4] |v| = 5 정규화된 v = [3/5, 4/5] = [0.6, 0.8] |정규화된 v| = 1 → 크기를 1로 만듦 → 방향은 유지, 크기만 조정 → AI에서 특징의 스케일을 맞출 때 사용

Part 6. 파이썬에서는?

파이썬으로 벡터를 어떻게 쓰는데?

NumPy 라이브러리를 써.

import numpy as np # 벡터 생성 a = np.array([1, 2, 3]) b = np.array([4, 5, 6]) # 벡터 덧셈 print(a + b) # [5 7 9] # 스칼라 곱 print(2 * a) # [2 4 6] # 내적 print(np.dot(a, b)) # 32 # 벡터 크기 print(np.linalg.norm(a)) # 3.7416... # 정규화 normalized = a / np.linalg.norm(a) print(normalized) # [0.267 0.534 0.801] print(np.linalg.norm(normalized)) # 1.0

보이지? NumPy로 벡터 연산이 엄청 쉬워.

AI 코딩할 때는 거의 항상 NumPy를 써.

핵심 정리

  • 벡터는 순서가 있는 숫자들의 집합 - [1, 2, 3] 형태
  • 여러 특징을 한 번에 표현 - 사과 = [색, 크기, 당도]
  • 내적이 가장 중요 - 유사도 측정의 핵심 도구
  • 모든 데이터가 벡터로 변환 - 이미지, 텍스트, 음성 전부
  • 차원 vs 크기 - 차원은 숫자 개수, 크기는 원점에서의 거리
  • NumPy로 쉽게 구현 - AI 코딩의 필수 라이브러리
