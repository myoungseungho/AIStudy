TOPIC 15

왜 수식이 이렇게 복잡해 보여?

Q. 시그마, 델타 기호는 뭐야?

Part 1. 시그마 (Σ)

Σ 기호가 뭐야?

합을 나타내는 기호

그냥 "전부 더하기"야.

시그마 (Summation): n Σ x_i = x_1 + x_2 + x_3 + ... + x_n i=1 읽기: "i가 1부터 n까지 x_i를 모두 더하기" 예시: 5 Σ i = 1 + 2 + 3 + 4 + 5 = 15 i=1 3 Σ i² = 1² + 2² + 3² = 1 + 4 + 9 = 14 i=1 코드로: sum = 0 for i in range(1, n+1): sum += x[i] → Σ는 그냥 for 루프!

Part 2. 델타 (Δ, δ)

델타는 뭐야?

변화량을 나타내는 기호

Δ (대문자 델타) = 큰 변화: Δx = x_2 - x_1 예시: x_1 = 5, x_2 = 8 Δx = 8 - 5 = 3 δ (소문자 델타) = 작은 변화: δx = 매우 작은 변화 → 미분에서 사용 ∂ (편미분 기호): ∂f/∂x = "f를 x로 편미분" → 델타와 비슷하게 생김 → 하지만 의미는 다름!

Part 3. 자주 나오는 기호들

다른 기호들도 알려줘

AI 논문에서 자주 보는 것들:

∇ (나블라, Nabla): ∇f = 경사도(gradient) = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ] → 모든 편미분을 모은 벡터 ∏ (파이, Product): n ∏ x_i = x_1 × x_2 × x_3 × ... × xₙ i=1 → Σ는 더하기, ∏는 곱하기 ∈ (원소): x ∈ A "x는 A의 원소" x ∈ ℝ "x는 실수" ∀ (모든): ∀x "모든 x에 대해" ∃ (존재): ∃x "어떤 x가 존재한다" ≈ (약): π ≈ 3.14 "파이는 약 3.14" ∝ (비례): y ∝ x "y는 x에 비례"

Part 4. 그리스 문자

알파, 베타, 감마... 왜 이렇게 많아?

관습적으로 의미가 정해져있어:

기호 이름 AI에서 의미
α 알파 학습률 (learning rate)
β 베타 모멘텀 계수, 정규화 파라미터
γ 감마 할인 계수, 스케일 파라미터
δ 델타 변화량, 오차
ε 엡실론 매우 작은 수 (1e-8)
θ 세타 파라미터 (가중치)
λ 람다 정규화 강도
μ 뮤 평균 (mean)
σ 시그마 표준편차 (std)
ρ 로 상관계수
π 파이 정책 (policy), 원주율

Part 5. 복잡한 수식 읽기

실제 수식은 어떻게 읽어?

단계별로 쪼개서 읽어:

예시 1: MSE n L = (1/n) Σ (y_i - ŷ_i)² i=1 읽기: 1. Σ: "합" 2. i=1부터 n까지: "모든 데이터에 대해" 3. (y_i - ŷ_i)²: "정답과 예측의 차이를 제곱" 4. (1/n): "n으로 나눠서 평균" → "모든 데이터의 오차 제곱을 평균낸 것" 예시 2: Softmax e^(x_i) σ(x)_i = ───────── Σ e^(x_j) j 읽기: 1. 분자: e^(x_i) "i번째 값을 지수로" 2. 분모: Σ e^(x_j) "모든 값을 지수로 한 후 합" 3. 나누기: "비율로 만들기" → "i번째 확률 = i번째 값 / 전체 합" 예시 3: 경사하강법 θ_new = θ_old - α ∇L(θ) 읽기: 1. θ: "파라미터" 2. ∇L(θ): "손실 함수의 경사도" 3. α: "학습률" 4. 마이너스: "반대 방향으로" → "파라미터 = 현재값 - 학습률×기울기"

Part 6. 팁: 수식을 코드로

수식을 코드로 바꾸는 팁은?

패턴을 알면 쉬워:

패턴 매칭: 수식 코드 ───────────────────────────────── Σ x_i sum(x) ∏ x_i prod(x) ∂f/∂x gradient ∇f torch.autograd x_i x[i] x̄ (x 평균) mean(x) ||x|| norm(x) x^T x.T (transpose) x ⊙ y x * y (element-wise) x · y dot(x, y) AB A @ B (matrix mult) 실제 예시: 수식: n Σ (x_i - μ)² i=1 코드: sum((x - mean(x)) ** 2) 또는: ((x - x.mean()) ** 2).sum()

핵심 정리

  • Σ = 합 - for 루프로 더하기
  • Δ = 변화량 - 차이
  • ∇ = 경사도 - 모든 편미분
  • 그리스 문자는 관습 - α=학습률, μ=평균 등
  • 쪼개서 읽기 - 한 번에 다 이해하려 하지 말고
  • 코드로 변환 - 수식을 코드로 바꾸면 이해 쉬움
