TOPIC 11

통계는 왜 배워야 해?

Q. 평균이랑 분산이 AI에 어떻게 쓰여?

Part 1. 평균(Mean)

평균은 알잖아. AI랑 무슨 상관이야?

데이터의 중심을 파악하려고

평균: μ = (Σx_i) / n 예시: 데이터: [2, 4, 6, 8, 10] 평균 = (2+4+6+8+10)/5 = 30/5 = 6 AI에서: 1. 배치 정규화 → 각 층의 출력을 평균 0으로 조정 2. 평균 손실 계산 → MSE = 평균 제곱 오차 3. 데이터 중심화 → 학습 전에 평균 빼기

Part 2. 분산(Variance)과 표준편차

분산은 뭐야?

데이터가 평균에서 얼마나 퍼져있는지

분산: σ² = Σ(x_i - μ)² / n 표준편차: σ = √분산 예시 1 (분산 작음): 데이터: [5, 6, 6, 7] 평균 = 6 분산 = ((5-6)² + (6-6)² + (6-6)² + (7-6)²) / 4 = 0.5 → 데이터가 뭉쳐있음 예시 2 (분산 큼): 데이터: [0, 2, 10, 12] 평균 = 6 분산 = ((0-6)² + (2-6)² + (10-6)² + (12-6)²) / 4 = 26 → 데이터가 널리 퍼져있음 AI에서: 1. 배치 정규화 → 분산을 1로 조정 2. 가중치 초기화 → 분산을 고려해서 초기값 설정 3. 데이터 정규화 → (x - μ) / σ

Part 3. 배치 정규화(Batch Normalization)

배치 정규화가 뭐길래 자주 나와?

딥러닝의 핵심 기법

각 층의 출력을 정규화해.

문제: 층1 출력: [100, 200, 150, ...] 평균=150, 분산=큼 층2 출력: [0.1, 0.3, 0.2, ...] 평균=0.2, 분산=작음 → 층마다 데이터 분포가 달라짐 → 학습이 불안정해짐 해결: Batch Normalization 1. 배치 평균 계산: μ 2. 배치 분산 계산: σ² 3. 정규화: x_norm = (x - μ) / σ 4. 스케일/시프트: y = γ×x_norm + β 결과: → 평균 0, 분산 1로 표준화 → 학습이 안정적이고 빨라짐 → 학습률을 높게 설정 가능

Part 4. 공분산과 상관계수

공분산은 뭐야?

두 변수가 함께 변하는 정도

공분산: Cov(X, Y) = Σ(x_i - μ_x)(y_i - μ_y) / n 해석: - 양수: X 증가 → Y도 증가 (양의 상관) - 음수: X 증가 → Y 감소 (음의 상관) - 0: 상관없음 상관계수 (정규화된 공분산): r = Cov(X, Y) / (σ_x × σ_y) → -1 ~ 1 사이 → -1: 완전 음의 상관 → 0: 무관 → 1: 완전 양의 상관 AI에서: 1. 특징 선택 → 상관이 높은 특징은 하나만 2. PCA (주성분 분석) → 공분산 행렬 사용 3. 데이터 전처리 → 다중공선성 제거

Part 5. 실전 사용

코드로는 어떻게 써?

NumPy와 PyTorch 예시:

import numpy as np # 평균과 분산 data = np.array([2, 4, 6, 8, 10]) mean = np.mean(data) # 6.0 var = np.var(data) # 8.0 std = np.std(data) # 2.828 # 정규화 normalized = (data - mean) / std print(normalized) # [-1.41 -0.71 0. 0.71 1.41] # 배치 정규화 (PyTorch) import torch import torch.nn as nn bn = nn.BatchNorm1d(num_features=10) x = torch.randn(32, 10) # 배치 32, 특징 10 output = bn(x) print(x.mean()) # ~0 (랜덤) print(output.mean()) # ~0 (정규화됨) print(output.std()) # ~1 (정규화됨)

핵심 정리

  • 평균 = 중심 - 데이터의 대표값
  • 분산 = 퍼진 정도 - 데이터의 변동성
  • 배치 정규화 - 평균 0, 분산 1로 표준화
  • 공분산 = 함께 변하는 정도 - 두 변수의 관계
  • 정규화는 필수 - AI 학습의 안정성과 속도 향상
  • 통계로 데이터 이해 - 전처리와 분석의 기초
