TOPIC 12

정규분포가 왜 중요해?

Q. 종 모양 그래프가 왜 자주 나와?

Part 1. 정규분포가 뭔데?

정규분포(Normal Distribution)가 뭐야?

자연에서 가장 흔한 확률 분포

종 모양(bell curve)이야.

정규분포: ╱‾╲ ╱ ╲ ╱ ╲ ___╱ ╲___ μ 특징: 1. 평균(μ) 중심으로 대칭 2. 평균에서 멀수록 확률 낮음 3. 표준편차(σ)가 퍼진 정도 결정 공식: f(x) = (1/(σ√(2π))) × e^(-(x-μ)²/(2σ²)) → 복잡해 보이지만 외울 필요 없음 → NumPy가 계산해줌 예시: 사람 키, 시험 점수, 측정 오차 → 대부분 정규분포를 따름

Part 2. 68-95-99.7 규칙

정규분포의 특별한 점은?

예측 가능한 패턴이 있어.

68-95-99.7 규칙: 68%: μ ± 1σ 범위 95%: μ ± 2σ 범위 99.7%: μ ± 3σ 범위 예시: 평균 = 170cm, 표준편차 = 10cm 68% 사람: 160~180cm 95% 사람: 150~190cm 99.7% 사람: 140~200cm → 200cm 넘으면 상위 0.15% (매우 드뭄)

Part 3. AI에서 어떻게 쓰여?

AI에서는 어디에 쓰이는 거야?

정말 많은 곳에 써:

1. 가중치 초기화 가중치를 정규분포로 랜덤 초기화: W ~ N(0, 0.01) → 평균 0, 표준편차 0.01 → 너무 크거나 작은 값 방지 2. 노이즈 추가 데이터 증강: x_noisy = x + ε, ε ~ N(0, σ²) → 정규분포 노이즈 추가 → 모델을 더 견고하게 3. 배치 정규화 가정 BN은 데이터가 정규분포를 따른다고 가정 → 평균 0, 분산 1로 정규화 4. VAE (Variational Autoencoder) 잠재 공간을 정규분포로 모델링 → 생성 모델의 핵심 5. 베이지안 딥러닝 가중치를 확률 분포로 표현 → 불확실성 추정

Part 4. 표준 정규분포

표준 정규분포는 뭐야?

평균 0, 분산 1인 정규분포

Z ~ N(0, 1)로 표기.

정규화 (Normalization): 원본: X ~ N(μ, σ²) 표준화: Z = (X - μ) / σ → Z ~ N(0, 1) 왜 중요? 1. 비교 가능 키와 몸무게를 같은 척도로 비교 2. 표준 표 사용 가능 확률 계산이 쉬워짐 3. AI 학습 안정화 모든 특징을 같은 스케일로

Part 5. 코드 예시

파이썬에서는 어떻게 써?

NumPy와 PyTorch 예시:

import numpy as np import matplotlib.pyplot as plt # 정규분포 샘플링 samples = np.random.normal(loc=0, scale=1, size=1000) # loc=평균, scale=표준편차 # 히스토그램 plt.hist(samples, bins=30) plt.show() # PyTorch에서 가중치 초기화 import torch import torch.nn as nn class Model(nn.Module): def __init__(self): super().__init__() self.fc = nn.Linear(10, 5) # Xavier 초기화 (정규분포 기반) nn.init.xavier_normal_(self.fc.weight) # He 초기화 (정규분포 기반) nn.init.kaiming_normal_(self.fc.weight) # 정규분포 확률 계산 from scipy import stats # Z ~ N(0, 1) prob = stats.norm.cdf(1.96) # P(Z ≤ 1.96) print(prob) # 0.975 (97.5%)

핵심 정리

  • 정규분포 = 종 모양 - 자연에서 가장 흔함
  • 평균과 분산으로 결정 - N(μ, σ²)
  • 68-95-99.7 규칙 - 예측 가능한 패턴
  • 가중치 초기화에 사용 - 학습 안정화
  • 표준화의 기준 - 평균 0, 분산 1로
  • AI의 핵심 가정 - 많은 기법의 기반
