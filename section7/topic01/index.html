TOPIC 01

강화학습은 뭐가 다를까?

Q. 레이블 없이 어떻게 배워?

Part 1. 지금까지 배운 학습 방식

지금까지 배운 머신러닝, 딥러닝이랑 강화학습은 뭐가 달라?

좋은 질문이야. 일단 지금까지 배운 걸 정리해보자.

지도학습은 "정답"이 있는 데이터로 배웠지?

비지도학습은 정답 없이 패턴을 찾았고.

지도학습 예시: 고양이 사진 1000장 + 레이블 "고양이" 개 사진 1000장 + 레이블 "개" → 이 데이터로 "분류기"를 학습 → 새로운 사진을 보고 고양이/개 판단 비지도학습 예시: 동물 사진 1만 장 (레이블 없음) → 비슷한 특징끼리 그룹화 → "이 그룹은 털이 있네", "저 그룹은 날개가 있네"

근데 강화학습은 완전히 다른 방식이야.

정답도 없고, 데이터도 없어.

대신 "경험"으로 배워.

Part 2. 강화학습은 어떻게 배워?

경험으로 배운다는 게 무슨 뜻이야?

게임으로 설명할게.

슈퍼마리오를 처음 해보는 AI 1회차: → 오른쪽으로 걸어감 → 굼바를 밟지 못하고 부딪힘 → 죽음 (보상: -100) 2회차: → 점프 버튼을 눌러봄 → 굼바를 밟아서 처치함 → 점수 획득! (보상: +100) 3회차: → 점프로 굼바를 피하는 게 좋다는 걸 알게 됨 → 더 멀리 가봄 → 코인 발견 (보상: +50) ... 1000회차: → 어떤 상황에서 어떻게 행동해야 하는지 학습 → 게임 클리어!

이해됐어?

강화학습은 "시행착오"로 배워.

누가 정답을 알려주지 않아. 스스로 해보고, 피드백을 받고, 개선하는 거지.

Part 3. 세 가지 학습 방식 비교

그럼 세 가지를 정확히 어떻게 구분해?

표로 정리하면 이해하기 쉬워.

학습 방식 데이터 학습 방법 예시
지도학습 입력 + 정답 레이블 정답을 맞추도록 학습 이미지 분류, 번역
비지도학습 입력만 (레이블 없음) 패턴/구조 찾기 클러스터링, 추천
강화학습 환경과의 상호작용 보상을 최대화 게임 AI, 로봇

핵심 차이:

지도학습 = "이게 정답이야" (선생님이 가르침)

비지도학습 = "뭔가 패턴을 찾아봐" (스스로 탐구)

강화학습 = "좋은 행동을 하면 보상을 줄게" (경험으로 배움)

Part 4. 왜 강화학습이 필요해?

지도학습만 쓰면 안 돼?

정답이 없는 문제들이 있거든.

정답이 없는 문제들: 바둑: → 이 수를 두는 게 "정답"인지 아무도 몰라 → 게임 끝나고 이겼는지 졌는지만 알 수 있음 → 수백 수를 두는 동안 어떤 수가 좋았는지 불명확 로봇 보행: → "왼발을 15도 각도로 올리는 게 정답"? → 정답 레이블을 만들 수가 없어 → 그냥 넘어지지 않고 걸으면 성공 자율주행: → 수백만 가지 상황에서의 "정답" 운전을 정의할 수 없음 → 안전하게 목적지에 도착하면 성공

이해됐지?

명확한 정답이 없는데, 목표는 있는 문제에 강화학습을 써.

"이기기", "걷기", "목적지 도착" 같은 목표를 달성하는 방법을 스스로 찾는 거야.

Part 5. 강화학습으로 뭘 만들었어?

실제로 강화학습이 성공한 사례는?

엄청 많아. 대표적인 것들만 보자.

강화학습 성공 사례: 1. 알파고 (2016) → 바둑에서 인간 챔피언 이김 → 수천만 번의 자가 대국으로 학습 2. OpenAI Five (2018) → Dota 2 게임에서 프로게이머 팀 격파 → 180년치 게임 시간을 시뮬레이션으로 학습 3. Atari 게임 (DeepMind) → 49개 게임을 하나의 알고리즘으로 학습 → 일부 게임에서 인간 수준 이상 달성 4. 로봇 제어 → 로봇 팔이 물체 잡는 법 학습 → 보스턴 다이내믹스 로봇의 균형 제어 5. 추천 시스템 → YouTube 영상 추천 → 사용자가 오래 볼 영상을 추천하도록 학습

공통점이 보이지?

모두 "정답"을 정의할 수 없는 문제들이야.

대신 "이기면 좋다", "오래 보면 좋다" 같은 목표만 있는 거지.

Part 6. 핵심 아이디어 정리

그럼 강화학습을 한 문장으로 요약하면?

"보상을 최대화하는 행동을 학습하는 것"

강화학습 작동 방식: 1. AI가 행동을 함 → 예: 게임에서 오른쪽으로 이동 2. 환경이 반응함 → 예: 코인을 먹거나, 적에게 부딪힘 3. 보상을 받음 → 코인 먹음: +10점 → 적에게 부딪힘: -100점 4. AI가 학습함 → "오른쪽으로 가는 게 이번엔 나빴네" → "다음엔 다른 행동을 해봐야겠어" 5. 반복 → 수천, 수만 번 반복하면서 최적의 행동 패턴 학습

이게 강화학습의 핵심이야.

시행착오(Trial and Error)를 통한 학습.

인간이 새로운 걸 배우는 방식과 비슷해.

📌 핵심 정리

  • 강화학습은 "경험"으로 배운다 - 정답 레이블이 필요 없음
  • 시행착오 학습 - 행동하고, 보상 받고, 개선하는 반복
  • 목표는 보상 최대화 - 어떻게든 보상을 많이 받는 방법 찾기
  • 정답 없는 문제에 적합 - 게임, 로봇, 추천 시스템 등
  • 대표 사례 - 알파고, OpenAI Five, 로봇 제어
