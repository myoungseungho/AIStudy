TOPIC 02

에이전트와 환경이 뭐야?

Q. 게임 캐릭터랑 게임 세계?

Part 1. 강화학습의 두 주인공

에이전트(Agent)랑 환경(Environment)이 뭐야?

강화학습을 이해하려면 이 두 개념을 꼭 알아야 해.

에이전트 = 학습하는 주체 (AI)

환경 = 에이전트가 상호작용하는 세계

비유로 이해하기: 게임으로 비유하면: - 에이전트 = 플레이어가 조종하는 캐릭터 - 환경 = 게임 세계 (맵, 적, 아이템, 규칙 등) 예: 슈퍼마리오 - 에이전트 = 마리오 - 환경 = 게임 맵, 굼바, 코인, 깃발 등 모든 것 예: 바둑 - 에이전트 = 바둑 AI (알파고) - 환경 = 바둑판 + 게임 규칙

이해됐어?

에이전트는 "행동"하고, 환경은 "반응"해.

Part 2. 에이전트와 환경은 어떻게 상호작용해?

둘이 어떻게 소통하는데?

순환 구조로 작동해.

강화학습 상호작용 사이클: ┌─────────────┐ │ 에이전트 │ │ (Agent) │ └──────┬──────┘ │ │ (1) 행동 (Action) ↓ ┌─────────────┐ │ 환경 │ │ (Environment)│ └──────┬──────┘ │ │ (2) 상태 + 보상 (State + Reward) ↓ ┌─────────────┐ │ 에이전트 │ └─────────────┘ 반복...
구체적인 예시로 설명해줘.

팩맨 게임으로 보자.

팩맨 게임 예시: 시작: 환경 → 에이전트에게 현재 상태 전달 "팩맨 위치: (5, 3), 유령 위치: (7, 3), 점수: 0" 에이전트 → 환경에 행동 전달 "오른쪽으로 이동" 환경 → 게임 규칙에 따라 상태 업데이트 팩맨이 오른쪽으로 이동함 점을 먹음 → 점수 +10 환경 → 에이전트에게 새로운 상태 + 보상 전달 "팩맨 위치: (6, 3), 유령 위치: (7, 3), 점수: 10" 보상: +10 에이전트 → 다시 행동 결정 "계속 오른쪽?" vs "방향 전환?" 반복...

이게 강화학습의 핵심 메커니즘이야.

에이전트와 환경이 끊임없이 주고받으면서 학습하는 거지.

Part 3. 에이전트는 정확히 뭘 해?

에이전트의 역할이 뭐야?

에이전트는 두 가지 일을 해.

1. 관찰 - 환경의 상태를 본다

2. 결정 - 어떤 행동을 할지 선택한다

에이전트의 사고 과정: 1단계: 관찰 "지금 상황이 어떤가?" → 현재 위치, 주변 장애물, 점수 등 파악 2단계: 결정 "어떤 행동을 해야 보상을 최대화할 수 있을까?" → 가능한 행동들 평가 → 최선의 행동 선택 3단계: 학습 "내가 한 행동이 좋았나, 나빴나?" → 보상을 보고 판단 → 정책(행동 전략) 업데이트

에이전트의 목표는 단 하나야.

"장기적으로 가장 많은 보상을 받는 것"

Part 4. 환경은 정확히 뭘 해?

환경의 역할은?

환경은 세 가지 일을 해.

1. 상태 제공 - 현재 상황 정보를 줌

2. 전이 - 행동에 따라 상태를 변경

3. 보상 - 행동에 대한 피드백을 줌

환경의 역할: 상태 제공: → "현재 게임 화면은 이렇습니다" → 에이전트가 의사결정에 필요한 정보 전달 전이 (Transition): → 에이전트가 "점프" 행동을 하면 → 물리 법칙에 따라 캐릭터 위치 변경 → 충돌 체크, 아이템 획득 등 처리 보상: → 좋은 일 발생: +보상 → 나쁜 일 발생: -보상 → 에이전트에게 피드백 제공
환경은 AI가 아니야?

아니야. 환경은 그냥 규칙에 따라 작동하는 시스템이야.

게임 엔진, 시뮬레이터, 또는 실제 세계일 수도 있어.

학습하는 건 에이전트만이야.

Part 5. 환경에는 어떤 종류가 있어?

환경도 종류가 다양해?

엄청 다양해. 환경의 특성에 따라 학습 난이도가 달라져.

환경 유형 설명 예시
결정적 환경 같은 행동 → 항상 같은 결과 체스, 바둑
확률적 환경 같은 행동 → 다른 결과 가능 주사위 게임, 포커
완전 관측 환경 모든 정보를 볼 수 있음 바둑판 전체 보임
부분 관측 환경 일부 정보만 볼 수 있음 포커 (상대 패 모름)
에피소드 환경 게임이 끝나는 환경 체스 게임 (승/패로 끝)
연속 환경 끝이 없는 환경 주식 거래, 추천 시스템

환경이 복잡할수록 학습이 어려워.

확률적이고, 부분 관측이고, 연속적인 환경일수록 난이도가 높아지는 거지.

Part 6. 실제 사례로 이해하기

실제 강화학습 시스템에서는 어떻게 구현돼?

구체적인 예시로 보자.

자율주행 자동차: 에이전트: - 자율주행 AI - 센서 데이터를 받아 운전 결정 환경: - 도로, 다른 차량, 신호등, 보행자 - 물리 법칙, 교통 법규 상호작용: 에이전트 → "핸들 5도 왼쪽으로" 환경 → 차량이 좌회전함 차선 유지 성공 → 보상 +1 다른 차량과 충돌 → 보상 -1000 로봇 팔: 에이전트: - 로봇 제어 AI - 관절 각도 조정 결정 환경: - 물리 세계 - 물체, 중력, 마찰력 상호작용: 에이전트 → "관절 1을 20도 회전" 환경 → 로봇 팔이 움직임 물체를 잡음 → 보상 +10 물체를 놓침 → 보상 -5

이해됐지?

에이전트와 환경은 분리된 개념이야.

에이전트만 학습하고, 환경은 그냥 반응해.

📌 핵심 정리

  • 에이전트 - 학습하는 주체, 행동을 결정하는 AI
  • 환경 - 에이전트가 상호작용하는 세계, 규칙에 따라 반응
  • 상호작용 순환 - 행동 → 상태 변화 + 보상 → 다음 행동
  • 에이전트의 목표 - 장기적 보상 최대화
  • 환경의 역할 - 상태 제공, 전이, 보상 제공
