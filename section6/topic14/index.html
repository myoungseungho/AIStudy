TOPIC 14

GPT는 BERT랑 뭐가 달라?

Q. 다음 단어 예측만 하는데 왜 강해?

Part 1. GPT의 핵심 차이

GPT(Generative Pre-trained Transformer)가 뭐야?

텍스트 생성에 특화된 모델이야.

BERT는 이해에 강하고, GPT는 생성에 강해.

핵심 차이: BERT (Bidirectional): "The cat [MASK] on the mat" ← 양방향으로 봄 → → 빈칸 채우기 전문 → 이해 능력 강함 GPT (Unidirectional): "The cat sat" → 다음은? → 왼쪽에서 오른쪽으로만 → → 다음 단어 예측 전문 → 생성 능력 강함 비유: BERT = 시험지 빈칸 채우기 → "한국의 수도는 ___이다" → "서울" GPT = 글쓰기 AI → "옛날에 한 마을에" → "살던 소년이 있었다..."
구분 BERT GPT
방향 양방향 (Bidirectional) 단방향 (Unidirectional)
학습 방법 MLM (빈칸 채우기) 다음 단어 예측
강점 문맥 이해, 분류 텍스트 생성
구조 Encoder만 사용 Decoder만 사용
활용 검색, 분류, QA 챗봇, 글쓰기, 번역

Part 2. GPT의 학습 방법

다음 단어 예측이 뭐야?

Auto-regressive Language Modeling이라고 해.

이전 단어들을 보고 다음 단어를 맞추는 거야.

학습 예시: 문장: "나는 오늘 학교에 갔다" 학습 단계별: 입력: "나는" 정답: "오늘" ← 다음 단어 맞추기 입력: "나는 오늘" 정답: "학교에" 입력: "나는 오늘 학교에" 정답: "갔다" → 매 단계 다음 단어 예측 학습 → 수십억 번 반복하면 글쓰기 학습! 실제 사용: "오늘 날씨가" (입력) → "좋네요" (예측) → "오늘 날씨가 좋네요" (생성) → "오늘 날씨가 좋네요 산책하기" (계속 생성) → "오늘 날씨가 좋네요 산책하기 좋은 날이에요" (완성!)
그게 왜 강력해?
GPT 학습의 장점: 1. 무한한 데이터 모든 텍스트가 학습 데이터 "오늘은 화요일이다" 한 문장에서: - "오늘은" → "화요일이다" - "오늘은 화요일이다" → ? → N개 학습 샘플 생성! 2. 라벨 불필요 정답 = 다음 단어 → 별도 라벨링 불필요 → 인터넷의 모든 텍스트 활용 3. 생성 능력 자연스럽게 습득 다음 단어 계속 예측 = 텍스트 생성 → 별도 생성 학습 불필요 4. 확장성 데이터 많을수록, 모델 클수록 성능 향상 → GPT-1: 1억 파라미터 → GPT-2: 15억 파라미터 → GPT-3: 1750억 파라미터 → GPT-4: 추정 1조+ 파라미터

Part 3. GPT의 진화

GPT가 계속 발전했어?
GPT 시리즈: GPT-1 (2018, OpenAI) - 1.17억 파라미터 - BookCorpus 학습 - 개념 증명 수준 GPT-2 (2019) - 15억 파라미터 - WebText (800만 문서) - "너무 위험해서 공개 못 함" (당시) - 나중에 전체 공개 GPT-3 (2020) - 1750억 파라미터 - 45TB 텍스트 학습 - Few-shot Learning 가능 - API로만 제공 ChatGPT (2022) - GPT-3.5 기반 - RLHF (인간 피드백) 학습 - 대화에 최적화 - 10억 사용자 돌파 GPT-4 (2023) - 크기 비공개 (추정 1조+) - 멀티모달 (텍스트+이미지) - 더 정확, 덜 헛소리 - 현재 최강
성능이 얼마나 좋아졌어?
성능 비교: 텍스트 생성 품질: GPT-1: 문법 OK, 의미 애매 GPT-2: 몇 문장까지 일관성 GPT-3: 수백 단어도 일관성 GPT-4: 수천 단어 에세이 수준 시험 성능: GPT-3: 미국 변호사 시험 하위 10% GPT-4: 미국 변호사 시험 상위 10%! GPT-3: SAT 점수 1260점 GPT-4: SAT 점수 1410점 (상위 15%) → 사람 수준에 근접 창의성: GPT-3: 시, 소설 작성 가능 GPT-4: 미묘한 유머, 스타일 모방 가능

Part 4. Few-Shot Learning

Few-Shot Learning이 뭐야?

예시 몇 개만 보고 새 태스크를 학습하는 거야.

GPT-3부터 가능해진 능력이지.

Few-Shot 예시: 번역 태스크 (학습 없이!): 입력: 영어: "Hello" → 한국어: "안녕하세요" 영어: "Thank you" → 한국어: "감사합니다" 영어: "I love you" → 한국어: GPT-3 출력: "나는 너를 사랑해" → 예시 2개만으로 번역 패턴 파악! → Fine-tuning 없이 바로 사용 감정 분석: 입력: "This movie is great!" → Positive "Terrible film" → Negative "Amazing performance" → GPT-3 출력: Positive → 예시로 태스크 이해! 종류: - Zero-Shot: 예시 0개 - One-Shot: 예시 1개 - Few-Shot: 예시 2~5개 → GPT-3 이전엔 불가능했던 능력

Part 5. BERT vs GPT 선택

언제 BERT 쓰고 언제 GPT 써?
BERT 사용: ✅ 문서 분류 스팸 메일 필터링 감정 분석 주제 분류 ✅ 검색 엔진 쿼리-문서 매칭 유사도 계산 ✅ 질의응답 (이해) 문서에서 답 찾기 정보 추출 ✅ Named Entity Recognition 인명, 지명 찾기 장점: - 양방향 → 문맥 이해 최고 - 빠름 - Fine-tuning 쉬움 GPT 사용: ✅ 텍스트 생성 글쓰기 이메일 작성 코드 생성 ✅ 대화 시스템 챗봇 고객 서비스 ✅ 번역 언어 간 변환 ✅ 요약 긴 글 → 짧은 요약 장점: - 생성 품질 최고 - Few-shot 가능 - 유연함 현실: → 이해 + 생성 둘 다 필요하면? → GPT-4 사용 (둘 다 잘함) → 요즘은 GPT 계열이 대세

다음 토픽에서 LLM(거대 언어 모델)에 대해 배워볼게!

핵심 정리

  • GPT - 다음 단어 예측으로 학습, 텍스트 생성에 특화
  • 단방향 - 왼쪽→오른쪽만 보지만 생성에는 더 유리
  • 진화 - GPT-1 → GPT-4, 파라미터 1억 → 1조+
  • Few-Shot - 예시 몇 개만으로 새 태스크 수행 가능
  • ChatGPT - GPT-3.5 기반, RLHF로 대화 최적화
