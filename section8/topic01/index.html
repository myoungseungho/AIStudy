TOPIC 01

추천 시스템은 어떻게 작동해?

Q. 넷플릭스는 내 취향을 어떻게 알아?

Part 1. 넷플릭스는 어떻게 내 취향을 알아?

넷플릭스를 켜면 딱 내가 좋아할 만한 게 떠. 어떻게 알고 그래?

추천 시스템(Recommendation System) 덕분이야.

넷플릭스, 유튜브, 쿠팡, 스포티파이... 다 이걸 써.

너의 행동 패턴을 보고, "이 사람은 이런 걸 좋아하겠네"라고 예측하는 거지.

그냥 내가 본 것만 추천하는 거 아니야?

그것보다 훨씬 복잡해.

너가 본 것뿐만 아니라:

- 시청 시간 (3분 보고 껐어? 끝까지 봤어?)

- 평점, 좋아요, 저장

- 같은 취향을 가진 다른 사람들의 데이터

- 시청한 시간대, 요일, 디바이스

이 모든 걸 종합해서 판단해.

비유: 서점 주인이 손님을 관찰한다고 생각해봐. "이 손님은 추리 소설 코너에서 오래 머물렀네." "이전에 히가시노 게이고 책을 샀었지." "비슷한 취향의 다른 손님들은 아가사 크리스티도 좋아하더라." → "아가사 크리스티 신간 나왔는데 관심 있으세요?" 추천 시스템도 똑같아. 너의 행동을 관찰하고, 패턴을 찾아서, 예측하는 거지.

Part 2. 협업 필터링: 비슷한 사람들의 데이터 활용

비슷한 취향을 가진 사람들의 데이터를 어떻게 써?

이걸 협업 필터링(Collaborative Filtering)이라고 해.

핵심 아이디어: "취향이 비슷한 사람들끼리는 앞으로도 비슷한 걸 좋아할 거야"

예시: [넷플릭스 데이터] 너: 오징어게임 ⭐⭐⭐⭐⭐, 이상한 변호사 우영우 ⭐⭐⭐⭐⭐ 사람A: 오징어게임 ⭐⭐⭐⭐⭐, 우영우 ⭐⭐⭐⭐⭐, 더 글로리 ⭐⭐⭐⭐⭐ → A랑 너의 취향이 비슷하네? → A가 본 "더 글로리"를 너한테 추천! 이게 협업 필터링이야.
그럼 다른 사람들 데이터를 계속 봐야 하는 거네?

맞아. 수백만 명의 데이터를 실시간으로 분석해서 추천해.

그래서 넷플릭스 추천이 점점 정확해지는 거야.

Part 3. 콘텐츠 기반 필터링: 아이템 자체를 분석

협업 필터링만 쓰는 건 아니지?

맞아. 콘텐츠 기반 필터링(Content-Based Filtering)도 있어.

이건 콘텐츠 자체의 특징을 분석해서 추천하는 거야.

예시: [오징어게임 특징] - 장르: 스릴러, 서바이벌 - 분위기: 긴장감, 반전 - 출연진: 이정재, 박해수 - 감독: 황동혁 - 키워드: 게임, 돈, 배신 → 비슷한 특징을 가진 작품 찾기 → "킹덤", "스위트홈" 추천 이게 콘텐츠 기반 필터링이야.
어느 게 더 좋아?

각각 장단점이 있어.

협업 필터링

✅ 장점: 의외의 추천 가능 (내가 몰랐던 장르 발견)

❌ 단점: 신작은 추천 어려움 (데이터가 없으니까)

콘텐츠 기반 필터링

✅ 장점: 신작도 바로 추천 가능

❌ 단점: 비슷한 것만 계속 추천 (새로운 발견 어려움)

그래서 실제로는 둘 다 섞어서 써! 이걸 하이브리드(Hybrid) 방식이라고 해.

Part 4. 딥러닝은 어떻게 추천을 더 잘해?

요즘은 AI를 쓴다면서? 더 똑똑해?

맞아. 딥러닝 추천 시스템은 더 복잡한 패턴을 찾아.

예시: 유튜브 추천 알고리즘 기존 방식: - 너가 본 영상의 카테고리 - 좋아요 누른 영상의 공통점 딥러닝 방식: - 썸네일을 봤을 때의 반응 - 제목 클릭률 - 시청 시간 패턴 (언제 재미없어서 껐나?) - 댓글, 공유 패턴 - 심지어 시청 속도 (1.5배속으로 봤나?) → 수백 개 변수를 동시에 고려해서 예측
그럼 더 정확한 거네?

정확하긴 한데, 부작용도 있어.

필터 버블(Filter Bubble)

→ 알고리즘이 너가 좋아하는 것만 계속 보여줘서, 새로운 걸 접할 기회가 줄어듦

에코 챔버(Echo Chamber)

→ 비슷한 의견만 반복해서 보게 돼서, 편향이 강화됨

그래서 가끔 의도적으로 다른 콘텐츠를 추천하기도 해. (탐색과 활용의 균형)

Part 5. 실생활에서 추천 시스템은 어디에 쓰여?

넷플릭스 말고 다른 곳에서도 써?

거의 모든 플랫폼이 써.

플랫폼 추천하는 것 활용 데이터
넷플릭스 영화, 드라마 시청 시간, 평점, 장르 선호도
유튜브 영상 클릭률, 시청 시간, 좋아요
스포티파이 음악, 플레이리스트 재생 횟수, 스킵 패턴, 시간대
쿠팡 상품 구매 이력, 검색, 장바구니
인스타그램 피드, 릴스 좋아요, 저장, 머문 시간

보이지? 추천 시스템은 현대 플랫폼 비즈니스의 핵심이야.

넷플릭스는 시청 시간의 80%가 추천 알고리즘에서 나온다고 해.

추천 시스템의 경제적 가치: 넷플릭스 → 추천 알고리즘의 가치: 연간 10억 달러 이상 → 이탈 방지 효과 (좋은 추천 = 구독 유지) 아마존 → 추천으로 인한 매출: 전체의 35% → "이 상품을 본 고객이 함께 본 상품" 유튜브 → 추천 영상 클릭률: 70% 이상 → 시청 시간의 대부분이 추천에서 발생 그래서 빅테크 기업들이 추천 시스템에 엄청난 투자를 하는 거야.

Part 6. 추천 시스템은 앞으로 어떻게 발전할까?

더 똑똑해질 수 있어?

당연하지. 트렌드를 몇 가지 알려줄게.

추천 시스템의 미래: 1. 실시간 맥락 인식 - 지금 기분, 날씨, 시간대, 함께 있는 사람까지 고려 - "주말 저녁, 가족과 함께" → 가족 영화 추천 - "평일 점심, 혼자" → 짧은 코미디 추천 2. 멀티모달 추천 - 텍스트, 이미지, 음성, 영상 모두 분석 - 썸네일 보고 클릭할지 예측 - 음악의 분위기를 시각적으로 분석 3. 대화형 추천 - "오늘 기분 좀 안 좋은데 뭐 볼까?" - → AI가 대화로 취향 파악 후 추천 4. 설명 가능한 추천 - "왜 이걸 추천했는지" 이유를 설명 - 투명성 증가, 신뢰 향상 5. 개인정보 보호 강화 - 데이터를 서버로 안 보내고 기기에서만 처리 - 프라이버시 중심 추천 (Federated Learning)

추천 시스템은 점점 더 정교해지고 있어.

하지만 동시에 윤리적 문제도 중요해지고 있지.

너의 취향을 정확히 예측하는 건 좋지만, 그게 조작으로 이어지면 안 되니까.

📌 핵심 정리

  • 추천 시스템 - 사용자 행동을 분석해서 좋아할 만한 것을 예측
  • 협업 필터링 - 비슷한 취향의 사람들 데이터 활용
  • 콘텐츠 기반 필터링 - 아이템 자체의 특징 분석
  • 딥러닝 추천 - 수백 가지 변수를 동시에 고려, 더 정교한 예측
  • 부작용 - 필터 버블, 에코 챔버 (같은 것만 계속 보게 됨)
  • 경제적 가치 - 넷플릭스 시청의 80%, 아마존 매출의 35%가 추천에서 발생
  • 미래 - 실시간 맥락, 멀티모달, 대화형, 설명 가능한 추천으로 진화 중
