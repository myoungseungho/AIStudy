TOPIC 03

자율주행은 어떻게 가능해?

Q. 카메라만으로 운전할 수 있어?

Part 1. 자율주행 차는 어떻게 운전하지?

테슬라는 사람 없이 운전해. 어떻게 가능한 거야?

센서 + AI 비전 + 의사결정 시스템이 합쳐진 거야.

사람이 운전할 때 뭘 하는지 생각해봐.

1. 눈으로 본다 (주변 차, 신호등, 사람)

2. 상황을 판단한다 (직진? 좌회전? 멈춤?)

3. 행동한다 (핸들, 가속, 브레이크)

자율주행도 똑같아.

자율주행 3단계: 1. 인식 (Perception) 센서로 주변 환경 파악 → "앞에 차가 있다", "신호등이 빨간불이다" 2. 판단 (Planning) AI가 다음 행동 결정 → "속도를 줄여야겠다", "차선을 바꿔야겠다" 3. 제어 (Control) 차량을 실제로 조작 → 브레이크 작동, 핸들 조작
센서는 뭘 써?

크게 3가지야.

센서 기능 장점 단점
카메라 이미지 인식 신호등, 표지판 읽기 날씨, 빛에 약함
라이다 (LiDAR) 레이저로 거리 측정 정확한 3D 지도 생성 비싸고, 비/눈에 약함
레이더 (Radar) 전파로 물체 감지 날씨에 강함, 속도 측정 해상도 낮음
GPS 위치 파악 지도 매칭 터널, 건물 사이에서 오차

보통 여러 센서를 조합해서 써. (센서 퓨전, Sensor Fusion)

카메라가 못 본 걸 라이다가 보고, 라이다가 헷갈리면 레이더가 확인하고.

Part 2. 테슬라 방식 vs 웨이모 방식

테슬라는 카메라만 쓴다면서? 라이다 없이 가능해?

이게 업계의 큰 논쟁이야.

테슬라 (일론 머스크): "사람도 눈만으로 운전하잖아? 카메라면 충분해"

웨이모 (구글): "안전을 위해 모든 센서 다 써야 해"

테슬라 방식 (Vision-Only): 센서: - 카메라 8개 (360도 커버) - 레이더 (최근 모델에서 제거) - 라이다 ✖️ (안 씀) 장점: ✅ 비용 절감 (라이다는 대당 수천만 원) ✅ 카메라 데이터가 더 풍부 (신호등, 표지판) ✅ 대규모 데이터 수집 (실제 차량 200만대+) 단점: ❌ 날씨, 야간에 약함 ❌ 거리 측정 부정확 (카메라는 2D) ❌ 안전성 논란 -------------------------------------------------- 웨이모 방식 (Multi-Sensor): 센서: - 카메라 다수 - 라이다 5개 - 레이더 다수 - 초음파 센서 장점: ✅ 안전성 높음 (센서 하나 고장나도 OK) ✅ 정확한 3D 인식 ✅ 날씨에 덜 민감 단점: ❌ 비용 비쌈 (대당 수천만~억 원) ❌ 복잡한 시스템 ❌ 대중화 어려움
누가 맞는 거야?

아직 정답은 없어.

테슬라는 비용과 확장성을 선택했고,

웨이모는 안전성과 신뢰성을 선택한 거지.

시간이 지나야 알겠지만, 테슬라가 AI 발전으로 카메라만으로도 충분해질 수 있다고 보는 거야.

Part 3. 자율주행 레벨: 0단계부터 5단계까지

완전 자율주행이랑 반자율주행이 다르다며?

맞아. SAE(미국 자동차공학회)가 6단계로 나눴어.

레벨 이름 설명 예시
Level 0 자동화 없음 모든 걸 사람이 함 일반 차량
Level 1 운전자 보조 크루즈 컨트롤 정도 속도 유지 기능
Level 2 부분 자율 특정 상황에서 자동 (손은 핸들에) 테슬라 오토파일럿, 현대 HDA
Level 3 조건부 자율 특정 조건에서 완전 자율 (개입 가능) 벤츠 Drive Pilot (독일)
Level 4 고도 자율 특정 지역에서 완전 자율 웨이모 택시 (미국 일부)
Level 5 완전 자율 어디서든 사람 개입 불필요 아직 없음
테슬라는 몇 레벨이야?

공식적으로는 Level 2야.

FSD (Full Self-Driving)라는 이름이지만, 완전 자율은 아니야.

운전자가 항상 핸들을 잡고 있어야 하고, 언제든 개입할 준비를 해야 해.

일론 머스크는 "곧 Level 4~5 될 거야"라고 하지만... 몇 년째 "내년에"라고 하고 있어.

현실 체크: 테슬라 FSD 베타: - 이름: "Full Self-Driving" (완전 자율주행) - 실제: Level 2 (운전자 항상 주시 필요) - 사고 시 책임: 100% 운전자 웨이모: - 샌프란시스코, 피닉스에서 Level 4 상용화 - 진짜 무인 택시 (운전석에 사람 없음) - 하지만 특정 지역만 가능 Level 5 (완전 자율): - 어디서든, 어떤 날씨든 OK - 핸들도 없는 차 - 아직 먼 미래 (10~20년?)

Part 4. AI는 자율주행에서 뭘 해?

센서로 본 걸 AI가 어떻게 처리해?

컴퓨터 비전(Computer Vision)으로 이미지를 분석해.

AI가 하는 일: 1. 객체 인식 (Object Detection) → "앞에 차가 있다", "사람이 건너간다" → YOLO, Faster R-CNN 같은 모델 사용 2. 차선 인식 (Lane Detection) → "내가 지금 2차선에 있다" → 차선 이탈 경고 3. 신호등/표지판 인식 → "빨간불이다", "정지 표지판이다" → OCR + 분류 모델 4. 깊이 추정 (Depth Estimation) → "저 차는 10m 앞에 있다" → 카메라 2개로 거리 계산 (스테레오 비전) 5. 경로 계획 (Path Planning) → "좌회전하려면 지금 차선 변경해야 해" → 강화학습으로 학습 6. 행동 예측 (Behavior Prediction) → "저 차가 끼어들 것 같다" → RNN, Transformer로 예측
이거 다 실시간으로 해?

그렇지. 초당 30~60프레임을 처리해야 해.

그래서 엄청난 컴퓨팅 파워가 필요해.

테슬라는 자체 AI 칩 (FSD Chip)을 만들었고,

엔비디아는 자율주행 전용 GPU를 팔아.

Part 5. 데이터 수집과 시뮬레이션

자율주행 AI는 어떻게 학습해?

실제 주행 데이터 + 시뮬레이션을 조합해.

테슬라의 데이터 전략: 1. 실제 차량 데이터 - 전 세계 200만대+ 테슬라가 데이터 수집 - 운전자가 개입한 순간 = 학습 데이터 - "여기서 AI가 실수했네?" → 서버로 전송 2. Shadow Mode (그림자 모드) - AI가 예측만 하고 실제로는 사람이 운전 - AI 예측 vs 사람 행동 비교 - 차이가 있으면 학습 3. 시뮬레이션 - 가상 환경에서 수억 마일 주행 - 사고 상황, 극단적 날씨 테스트 - 현실에서 하기 어려운 상황 학습 → 하루 수백만 마일치 데이터 수집
웨이모는?

웨이모는 시뮬레이션에 더 집중해.

"Carcraft"라는 가상 세계에서 테스트해.

현실에서 1마일 주행하면, 가상에서 1만 마일 테스트.

안전성을 최우선으로 하니까.

Part 6. 자율주행의 한계와 미래

완전 자율주행은 언제쯤 가능해?

생각보다 오래 걸릴 거야.

기술적 한계:

- Edge Case (극단적 상황) 처리 어려움

- 눈, 비, 안개에서 센서 성능 저하

- 사람의 비합리적 행동 예측 어려움 (갑자기 뛰어드는 사람)

법적/윤리적 문제:

- 사고 시 책임은 누구? (제조사? 소유자?)

- 트롤리 딜레마 (누구를 먼저 보호할 것인가?)

- 보험, 법규 정비 필요

자율주행의 미래: 단기 (2025~2030): - Level 3 대중화 (고속도로, 단순 구간) - Level 4 로봇택시 확대 (특정 도시) - 물류, 배송에 먼저 적용 중기 (2030~2040): - Level 4가 일반적 - 도심에서도 자율주행 택시 - 자율주행 전용 차선, 구역 장기 (2040~): - Level 5 등장 (완전 자율) - 핸들 없는 차량 상용화 - 차가 이동 수단이 아닌 "움직이는 공간"으로 하지만 예측은 어려워. 10년 전만 해도 "2020년엔 다 자율주행일 거야" 했는데...

자율주행은 점진적으로 발전할 거야.

갑자기 완벽한 Level 5가 나오는 게 아니라,

특정 상황에서부터 조금씩 확대되는 거지.

먼저 고속도로, 그 다음 도심, 마지막으로 비포장도로.

📌 핵심 정리

  • 3단계 구조 - 인식(센서) → 판단(AI) → 제어(액추에이터)
  • 센서 - 카메라, 라이다, 레이더, GPS 조합 (센서 퓨전)
  • 테슬라 vs 웨이모 - 카메라만 vs 모든 센서 (논쟁 중)
  • 자율주행 레벨 - Level 0~5 (현재 대부분 Level 2)
  • AI 역할 - 객체 인식, 차선 인식, 경로 계획, 행동 예측
  • 데이터 - 실제 주행 데이터 + 시뮬레이션으로 학습
  • 미래 - 점진적 발전 (특정 상황부터 확대), Level 5는 먼 미래
