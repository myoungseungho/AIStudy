TOPIC 05

분류와 회귀는 뭐가 달라?

Q. 고양이/개 구분이랑 집값 예측이 달라?

Part 1. 답이 "이름"이냐, "숫자"냐

분류(Classification)와 회귀(Regression)는 뭐가 다른 거야?

예측하는 답의 종류가 달라.

쉽게 말하면:

분류 (Classification): → 카테고리(이름)로 답하기 → "A냐 B냐?" 예시: Q: 이 동물은? A: "고양이" 또는 "개" Q: 이 메일은? A: "스팸" 또는 "정상" 회귀 (Regression): → 숫자로 답하기 → "얼마나?" 예시: Q: 이 집 가격은? A: "5억 2천만원" Q: 내일 기온은? A: "27.5도"

이해됐어?

분류는 "종류를 구분", 회귀는 "값을 예측"하는 거야.

Part 2. 학교 시험으로 비유하면?

아직 헷갈려. 좀 더 쉽게 설명해줘

학교 시험 문제로 비유해볼게.

분류 = 객관식 문제 Q: 한국의 수도는? ① 서울 ② 부산 ③ 대구 ④ 인천 → 정해진 선택지 중에서 고르기 → 딱 떨어지는 답 회귀 = 단답형/서술형 문제 Q: 서울에서 부산까지 거리는? A: 325km (숫자로 답) Q: 이 삼각형의 넓이는? A: 42.7cm² (계산된 값) → 연속된 값 중 하나 → 정확한 숫자

분류는 "선택", 회귀는 "계산"이라고 생각하면 돼.

Part 3. 한눈에 비교하기

표로 정리해줄 수 있어?

물론이지. 차이점을 명확하게 보여줄게.

구분 분류 (Classification) 회귀 (Regression)
출력 형태 카테고리 (이산적) 숫자 (연속적)
예시 답 "고양이", "개", "새" 5.2, 127.8, 3억
문제 유형 "어느 그룹인가?" "얼마인가?"
평가 방법 정확도, 정밀도, 재현율 오차 (MAE, RMSE)
실생활 예시 스팸 필터, 얼굴 인식 주가 예측, 날씨 예측

차이가 명확하지?

Part 4. 같은 데이터, 다른 문제

같은 데이터로도 분류와 회귀를 둘 다 할 수 있어?

그럼! 좋은 질문이야. 예시를 보여줄게.

집 데이터로 두 가지 문제 풀기: 데이터: 평수, 위치, 층수, 방 개수... [분류 문제로 풀기] Q: 이 집이 비싼 편인가, 저렴한 편인가? A: "비싼 집" 또는 "저렴한 집" → 2개 카테고리로 구분 [회귀 문제로 풀기] Q: 이 집의 정확한 가격은? A: "5억 2,300만원" → 구체적인 숫자 예측 어떤 게 더 유용할까? → 상황에 따라 다름! → 대충 비싼지만 알고 싶으면 → 분류 → 정확한 가격이 필요하면 → 회귀

같은 데이터라도 "어떤 질문을 하느냐"에 따라 분류/회귀가 결정돼.

Part 5. 분류에도 종류가 있어

분류는 다 똑같은 거야?

아니, 분류도 세분화할 수 있어.

분류의 종류: 1. 이진 분류 (Binary Classification) → 2개 중 하나 선택 예시: - 스팸 vs 정상 - 합격 vs 불합격 - 양성 vs 음성 2. 다중 분류 (Multi-class Classification) → 3개 이상 중 하나 선택 예시: - 고양이 vs 개 vs 새 vs 토끼 - A학점 vs B학점 vs C학점 vs D학점 vs F학점 - 액션 vs 로맨스 vs 코미디 vs 공포 3. 다중 레이블 분류 (Multi-label Classification) → 여러 개 동시 선택 가능 예시: 영화 장르: "액션 + 코미디 + 어드벤처" (한 영화가 여러 장르에 속할 수 있음)

이해됐지?

선택지가 몇 개냐, 하나만 고르냐 여러 개 고르냐에 따라 달라져.

Part 6. 언제 분류, 언제 회귀를 써?

내 문제가 분류인지 회귀인지 어떻게 알아?

간단한 질문으로 판단할 수 있어.

판단 기준: 질문: "예측할 값이 정해진 카테고리인가?" YES → 분류 예시: - "이 환자는 당뇨병인가?" (예/아니오) - "이 꽃은 무슨 종인가?" (장미/튤립/해바라기) - "이 문장의 감정은?" (긍정/부정/중립) NO → 회귀 예시: - "이 환자의 혈당 수치는?" (120mg/dL) - "이 꽃의 키는?" (32.5cm) - "이 문장의 긍정도는?" (0.87점) 애매한 경우: 나이 예측 → 분류: "10대/20대/30대..." (연령대) → 회귀: "27세" (정확한 나이) → 둘 다 가능! 목적에 맞게 선택

핵심은 "정답이 딱 떨어지는 카테고리냐, 연속된 숫자냐"야.

핵심 정리

  • 분류 = 카테고리 구분 - "고양이냐 개냐" 같은 이산적 답
  • 회귀 = 숫자 예측 - "5억 2천만원" 같은 연속적 값
  • 분류 종류 - 이진(2개), 다중(3개 이상), 다중 레이블(여러 개 동시)
  • 같은 데이터, 다른 문제 - 어떤 질문을 하느냐에 따라 달라짐
  • 평가 방법도 다름 - 분류는 정확도, 회귀는 오차로 평가
  • 선택 기준 - 답이 카테고리면 분류, 숫자면 회귀
