TOPIC 03

레이블이 뭐야?

Q. 데이터에 정답을 달아준다는 게?

Part 1. 레이블은 정답 스티커야

레이블(Label)이 뭐야? 왜 필요해?

레이블 = 데이터에 붙이는 정답 표시

옷 가게에서 옷에 가격표 붙이는 거 있지? 그거랑 똑같아.

비유: 도서관 책 분류 책 → 라벨(스티커) 붙이기 [해리포터] → "판타지" [코스모스] → "과학" [전쟁과 평화] → "문학" → 나중에 책 찾을 때 쉽게 구분 가능 AI 학습도 똑같아: [고양이 사진] → "고양이" [개 사진] → "개" [스팸 메일] → "스팸" → AI가 나중에 새로운 데이터를 분류할 수 있음

이해됐어?

레이블은 "이 데이터가 뭔지" 알려주는 정답이야.

Part 2. 레이블에도 종류가 있어

레이블은 다 똑같은 거 아니야?

아니야. 어떤 문제를 풀거냐에 따라 다르게 달아.

1. 분류 레이블 (카테고리) 이미지 분류: 고양이 사진 → "고양이" 개 사진 → "개" 새 사진 → "새" 감정 분석: "이 영화 최고!" → "긍정" "돈 아까워" → "부정" 2. 회귀 레이블 (숫자) 집값 예측: 집 정보 (평수, 위치...) → "5억 2천만원" 집 정보 (평수, 위치...) → "3억 8천만원" 날씨 예측: 날씨 데이터 → "27.5도" 날씨 데이터 → "15.3도" 3. 세밀한 레이블 (여러 정보) 객체 탐지: 이미지 → "고양이 (x:100, y:200, 크기:50x60)" 이미지 → "사람 (x:300, y:100, 크기:80x120)"

보이지?

문제에 맞게 적절한 형태로 레이블을 달아야 해.

Part 3. 레이블은 누가 달아?

레이블은 컴퓨터가 자동으로 달아줘?

아니, 사람이 직접 달아야 해.

그래서 엄청 힘들어.

실제 레이블링 작업: 고양이/개 분류 AI 만들기 1단계: 이미지 수집 → 고양이 사진 10,000장 → 개 사진 10,000장 2단계: 레이블링 (사람이 직접!) 사진1 보고 → "고양이" 클릭 사진2 보고 → "개" 클릭 사진3 보고 → "고양이" 클릭 ... (20,000번 반복) 3단계: 검수 → 실수로 잘못 달은 레이블 수정 → 애매한 사진 다시 확인 → 시간, 돈, 인력 엄청 듦
이거 사람이 다 해야 돼?

응. 그래서 실제 AI 프로젝트에서는:

1. 크라우드소싱 - 여러 사람에게 나눠서 시킴

2. 전문가 고용 - 의료 AI 같은 경우 전문의가 레이블링

3. 반자동화 - AI가 대충 달고, 사람이 검수

Part 4. 레이블이 틀리면 어떻게 돼?

레이블을 잘못 달면 뭐가 문제야?

AI가 잘못된 걸 배워.

쉬운 예시로 보여줄게.

잘못된 레이블의 영향: 데이터: 고양이 사진 → "개" (잘못됨!) 고양이 사진 → "개" (잘못됨!) 개 사진 → "고양이" (잘못됨!) AI 학습 결과: → 고양이를 개로 인식 → 개를 고양이로 인식 → 완전히 반대로 배움! 현실 사례: 의료 AI에서 레이블 실수 → 정상 X-ray를 "비정상"으로 레이블 → AI가 정상을 비정상으로 진단 → 잘못된 치료 가능성 자율주행에서 레이블 실수 → 사람을 "장애물"로 레이블 → AI가 사람을 단순 장애물로 인식 → 사고 위험!

이해됐지?

"쓰레기 데이터 넣으면, 쓰레기 결과 나온다"

이걸 GIGO (Garbage In, Garbage Out)라고 해.

Part 5. 좋은 레이블은 어떻게 만들어?

그럼 레이블을 잘 만들려면 어떻게 해야 해?

몇 가지 원칙이 있어.

좋은 레이블링 원칙: 1. 일관성 → 같은 기준으로 달기 → 사람마다 다르게 달면 안 됨 예시: 진돗개를 누구는 "개", 누구는 "늑대" ❌ 모두 "개"로 통일 ✓ 2. 정확성 → 애매하면 확실히 확인하고 달기 → 대충 찍어서 달면 안 됨 3. 완전성 → 필요한 정보를 빠짐없이 달기 이미지에 고양이 2마리 있으면 → 2마리 다 표시해야 함 4. 명확한 가이드라인 → "이런 경우는 이렇게 달기" 규칙 정해두기 예시: Q: 강아지가 멀리 있어서 흐릿하면? A: 50% 이상 보이면 "개", 아니면 "배경"

레이블 품질 = AI 성능이야.

아무리 좋은 알고리즘이어도, 레이블이 엉망이면 소용없어.

Part 6. 실제로는 어떤 도구를 써?

레이블을 일일이 손으로 달아?

아니, 레이블링 도구를 써.

대표적인 레이블링 도구: 이미지 레이블링: - Labelbox: 이미지에 박스 그리고 레이블 달기 - CVAT: 동영상 프레임마다 레이블링 - Roboflow: 객체 탐지용 레이블링 텍스트 레이블링: - Prodigy: 텍스트 감정, 카테고리 레이블링 - Amazon SageMaker: 아마존 제공 레이블링 서비스 크라우드소싱 플랫폼: - Amazon MTurk: 여러 사람에게 레이블 작업 의뢰 - Scale AI: 전문 레이블러들이 작업

현실에서는 이런 도구 + 사람 조합으로 레이블링해.

100% 자동은 불가능하고, 사람의 품질 관리가 꼭 필요해.

핵심 정리

  • 레이블 = 정답 표시 - 데이터에 "이게 뭔지" 알려주는 스티커
  • 종류가 다양 - 카테고리, 숫자, 위치 정보 등 문제에 맞게 달기
  • 사람이 직접 달아야 함 - 자동화 불가능, 시간과 비용 많이 듦
  • 품질이 중요 - 레이블이 틀리면 AI가 잘못 배움 (GIGO)
  • 일관성과 정확성 - 같은 기준으로 정확하게 달아야 함
  • 전문 도구 사용 - 레이블링을 돕는 다양한 도구 활용
