TOPIC 02

지도학습이 뭐야?

Q. 선생님이 있다는 뜻이야?

Part 1. 선생님이 있다는 뜻이 맞아

"지도학습(Supervised Learning)"이라는 이름이 이상해. 누가 지도한다는 거야?

사람이 정답을 알려준다는 뜻이야.

학교에서 선생님이 가르쳐주는 것처럼 말이야.

비유: 학교 수학 시험 선생님이 하는 일: 1. 문제를 주고 2. 풀게 하고 3. 정답을 알려주고 4. 틀린 부분을 고쳐줌 → 학생은 정답을 보면서 배움 지도학습도 똑같아: 1. 데이터(문제)를 주고 2. AI가 예측하게 하고 3. 정답을 알려주고 4. 틀린 부분을 수정함 → AI는 정답을 보면서 배움

차이가 뭐냐면,

지도학습에서는 모든 데이터에 정답이 달려 있어.

이 "정답"을 우리는 레이블(Label)이라고 불러.

Part 2. 지도학습은 어떻게 진행돼?

구체적으로 어떻게 학습하는 건데?

단계별로 보여줄게.

고양이/개 분류 AI 만들기 (지도학습) 1단계: 데이터 준비 (정답 포함!) 이미지1: 🐱 → 정답: "고양이" 이미지2: 🐶 → 정답: "개" 이미지3: 🐱 → 정답: "고양이" 이미지4: 🐶 → 정답: "개" ... (수천, 수만 개) 2단계: 학습 AI: "이 이미지는 뭘까? 개인 것 같은데..." 정답: "고양이야" AI: "아, 틀렸네. 수정해야겠다" → 이걸 수천 번 반복 3단계: 검증 새 이미지: 🐱 AI: "고양이다!" 정답: "맞아!" → 정확도가 높아질 때까지 반복

이해됐어?

핵심은 "정답을 알려주면서 학습한다"는 거야.

Part 3. 정답이 왜 필요한 거야?

정답 없이는 못 배워?

생각해봐. 너한테 외국어 단어 100개를 보여주는데, 뜻을 안 알려주면?

배울 수가 없잖아.

정답이 없을 때: 이미지: 🐱 AI: "이게 뭐지? 고양이? 개? 토끼?" → 확인할 방법이 없음 → 학습 불가능 정답이 있을 때: 이미지: 🐱 → 정답: "고양이" AI: "아, 이런 특징이 있으면 고양이구나" 이미지: 🐶 → 정답: "개" AI: "이런 특징이 있으면 개구나" → 차이를 배울 수 있음

정답이 있어야 "맞았는지 틀렸는지" 알 수 있고,

그래야 "어떻게 고쳐야 할지" 알 수 있어.

Part 4. 지도학습으로 뭘 할 수 있어?

지도학습으로 할 수 있는 게 뭐야?

크게 두 가지야.

1. 분류 (Classification) → 카테고리로 나누기 예시: - 스팸 메일인가? (스팸 vs 정상) - 고양이인가 개인가? (고양이 vs 개) - 긍정 리뷰인가 부정 리뷰인가? (긍정 vs 부정) 정답이 "카테고리" 2. 회귀 (Regression) → 숫자 값 예측하기 예시: - 내일 기온은? (25.3도) - 이 집값은? (5억 2천만원) - 이 학생의 시험 점수는? (85점) 정답이 "숫자"

둘 다 정답이 있다는 게 공통점이야.

정답의 형태만 다를 뿐이지.

Part 5. 지도학습의 문제점은?

그럼 지도학습이 완벽한 거 아냐?

큰 문제가 하나 있어.

정답을 달아주는 게 엄청 힘들어.

현실의 문제: 고양이/개 분류 AI 만들기 → 이미지 10만 장에 사람이 일일이 "고양이", "개" 표시 → 몇 달 걸림 의료 AI 만들기 → X-ray 사진 100만 장에 전문의가 "정상", "비정상" 표시 → 몇 년 걸림 → 시간, 돈, 인력이 엄청 필요함
그래서 어떻게 해?

두 가지 방법이 있어:

1. 크라우드소싱 - 여러 사람한테 나눠서 시킴 (아마존 MTurk)

2. 비지도학습 - 정답 없이 학습하는 방법 (다음 토픽에서 배움)

Part 6. 실제 지도학습 사례

실제로는 어디에 쓰여?

우리 주변 곳곳에 있어.

실전 지도학습 예시: 얼굴 인식 (Face ID) - 데이터: 내 얼굴 사진 여러 장 (다양한 각도, 조명) - 정답: "이게 주인이야" - 결과: 내 얼굴만 인식 음성 인식 (시리, 구글 어시스턴트) - 데이터: 사람들이 말한 음성 수백만 개 - 정답: 각 음성의 텍스트 - 결과: 말을 텍스트로 변환 번역 AI (구글 번역, 파파고) - 데이터: 한영 대역 문장 수백만 쌍 - 정답: "이 영어는 이 한국어야" - 결과: 자동 번역

보이지?

정답이 명확한 문제는 대부분 지도학습으로 해결해.

핵심 정리

  • 지도학습 = 정답을 주고 학습 - 모든 데이터에 레이블(정답)이 있음
  • 선생님 역할 - 사람이 정답을 알려주면서 AI를 가르침
  • 두 가지 종류 - 분류(카테고리 구분)와 회귀(숫자 예측)
  • 장점 - 정확한 학습 가능, 결과 예측 가능
  • 단점 - 정답 달기가 힘들고 비용이 많이 듦
  • 활용 사례 - 얼굴 인식, 음성 인식, 번역, 스팸 필터 등
