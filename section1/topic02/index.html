TOPIC 02

AI를 본 적 있어?

Q. ChatGPT, 시리, 알렉사... 이게 다 AI야?

Part 1. AI는 이미 네 주변에 있어

AI가 뭔지는 알겠는데, 나는 AI를 써본 적이 없는 것 같아.

착각이야.

너 이미 매일 AI 쓰고 있어.

무슨 소리야? 나 ChatGPT도 안 쓰는데?

ChatGPT만 AI가 아니야. 생각해봐:

너 어제 한 일: 아침: 스마트폰 알람 끄기 → "헤이 시리, 알람 꺼줘" ← AI 출근길: 넷플릭스/유튜브 추천 영상 봄 → "너가 좋아할 만한 영상" ← AI 점심: 검색하면서 자동완성 사용 → "검색어 자동 완성" ← AI 저녁: 인스타그램 스크롤 → "피드 정렬 알고리즘" ← AI 밤: 스팸 메일 필터링 → "이 메일은 스팸입니다" ← AI

보여? AI는 이미 네 일상 구석구석에 있어.

그냥 "AI입니다!"라고 광고를 안 해서 몰랐을 뿐이지.

Part 2. 보이지 않는 AI들

그럼 내가 모르는 사이에 AI가 뭘 하고 있는 거야?

엄청 많아. 몇 가지만 볼게:

서비스 AI가 하는 일 네가 경험하는 것
넷플릭스 시청 패턴 분석 → 추천 "지금 뜨는 콘텐츠"
유튜브 영상 내용 분석 → 추천 홈 화면 추천 영상
구글 포토 얼굴 인식 → 자동 분류 "사람별로 사진 모아보기"
스마트폰 카메라 피사체 인식 → 최적 모드 "야간 모드 자동 전환"
카카오톡 텍스트 분석 → 이모티콘 추천 입력창 위 추천 이모티콘
네이버 쇼핑 클릭 패턴 분석 → 추천 "이 상품은 어때요?"

이해됐어?

AI는 "ChatGPT처럼 대화하는 것"만 의미하지 않아.

패턴을 인식하고 예측하는 모든 시스템이 AI야.

Part 3. 그럼 시리도 ChatGPT랑 똑같아?

시리, 알렉사, 빅스비도 AI잖아. 근데 ChatGPT랑 느낌이 다른데?

맞아, 세대가 다른 거야.

구세대 AI (시리, 알렉사) 신세대 AI (ChatGPT, Gemini)
방식 명령어 기반 대화 기반
정해진 답변 O (미리 프로그래밍) X (실시간 생성)
복잡한 질문 잘 못함 잘함
맥락 이해 약함 강함
비유 리모컨 (버튼 눌러야 작동) 비서 (알아서 이해)
그럼 시리는 진짜 AI가 아닌 거야?

아니야, AI는 맞아.

다만 AI 기술의 수준이 다른 거지.

비유: 자동차도 세대가 있잖아? - 1990년대 자동차 = 수동 변속기 - 2020년대 자동차 = 자율주행 둘 다 "자동차"는 맞아. 근데 기술 수준이 다른 거지. AI도 마찬가지: - 2011년 시리 = 명령어 인식 AI - 2023년 ChatGPT = 생성형 AI 둘 다 "AI"는 맞아. 근데 기술 수준이 다른 거야.

Part 4. 왜 AI인 줄 몰랐을까?

그럼 왜 나는 이게 AI인 줄 몰랐어?

이유 1: 너무 자연스러워서

AI가 잘 작동하면, 그냥 "원래 그런가 보다" 생각하게 돼.

예시: 유튜브 추천 영상 → "아, 유튜브가 내 취향을 안다" → "AI가 학습했구나"라고 생각 안 함 구글 번역 → "오, 번역이 잘 되네" → "AI가 번역하는구나"라고 생각 안 함 얼굴 인식 잠금 해제 → "편하네" → "AI가 내 얼굴 인식하는구나"라고 생각 안 함

이유 2: "AI"라고 부르지 않아서

회사들이 굳이 "이건 AI입니다!"라고 광고하지 않아.

그냥 "스마트 추천", "자동 필터", "맞춤형" 같은 말을 쓰지.

이유 3: 결과만 보고 과정은 안 봐서

AI는 백그라운드에서 작동해.

너는 결과만 보지, AI가 뭘 했는지는 안 보는 거야.

Part 5. 직접 찾아보기

그럼 내가 쓰는 앱 중에 AI가 어디 있는지 어떻게 알아?

간단한 체크리스트:

AI가 있는지 확인하는 법: ✅ "추천"이라는 말이 나오는가? → 추천 = AI가 패턴 학습 ✅ "자동"이라는 말이 나오는가? → 자동 태그, 자동 분류 = AI가 인식 ✅ "검색"이 똑똑한가? → 오타도 알아듣기, 연관 검색어 = AI가 분석 ✅ "알림"이 정확한가? → "관심 있을 만한 소식" = AI가 예측 ✅ "필터"가 작동하는가? → 스팸 필터, 유해 콘텐츠 차단 = AI가 판단
그럼 거의 모든 앱에 AI가 있는 거네?

맞아. 2024년 기준으로 대부분의 앱에 AI가 있어.

AI가 없으면 경쟁에서 밀리기 때문이야.

AI 없는 앱 vs AI 있는 앱: 음악 앱 (AI 없음) → 내가 직접 플레이리스트 만듦 → 새 음악은 일일이 검색 음악 앱 (AI 있음) → "취향 맞춤 플레이리스트" 자동 생성 → "이 노래 어때?" 추천 → 당연히 AI 있는 앱이 이기겠지?

📌 핵심 정리

  • AI는 이미 일상에 있다 - 넷플릭스, 유튜브, 구글, 인스타그램 전부 AI 사용
  • 눈에 안 보일 뿐 - 백그라운드에서 작동해서 AI인 줄 모름
  • 세대 차이 - 시리(구세대) vs ChatGPT(신세대), 둘 다 AI지만 수준이 다름
  • 확인법 - "추천", "자동", "맞춤형" 같은 키워드가 있으면 AI
  • 대부분의 앱에 있다 - 2024년 기준, AI 없으면 경쟁력 없음
