TOPIC 08

데이터가 왜 중요한 거야?

Q. 데이터 없으면 AI 못 만들어?

Part 1. AI는 데이터를 먹고 자라

데이터가 왜 중요한 거야? 알고리즘만 좋으면 되는 거 아니야?

"데이터는 AI의 연료"야.

아무리 좋은 엔진(알고리즘)이라도 연료(데이터) 없으면 못 달려.

비유: 요리 좋은 요리사 (알고리즘) + 형편없는 재료 (데이터) = 맛없는 음식 평범한 요리사 + 신선한 재료 = 맛있는 음식 → 재료가 더 중요할 때도 많아 AI도 똑같아: 최신 알고리즘 + 100개 데이터 = 정확도 60% 평범한 알고리즘 + 10,000개 데이터 = 정확도 90% → "Garbage In, Garbage Out" → 쓰레기 데이터 넣으면 쓰레기 결과 나옴

Part 2. 데이터 양이 중요해

데이터가 얼마나 필요해?

많으면 많을수록 좋아.

데이터 양에 따른 성능: 고양이 인식 AI: 100장: 정확도 50% (동전 던지기 수준) 1,000장: 정확도 70% 10,000장: 정확도 85% 100,000장: 정확도 95% 1,000,000장: 정확도 98% ChatGPT: → 인터넷 전체 텍스트 학습 → 수조 개의 단어 → 수백 테라바이트 → 이 정도 데이터가 있어야 사람처럼 대화 가능
왜 많아야 해?

이유 1: 다양한 경우를 봐야 함

고양이도 여러 종류 있잖아? 페르시안, 샴, 코숏...

전부 봐야 "고양이"를 제대로 이해함.

이유 2: 예외 상황 대비

앉은 고양이, 누운 고양이, 점프하는 고양이...

다양한 포즈 전부 학습해야 인식 가능.

Part 3. 데이터 품질이 더 중요해

그럼 무조건 많기만 하면 돼?

아니야!

품질이 더 중요해.

나쁜 데이터 예시: 고양이 인식 AI를 만드는데... 문제 1: 잘못된 라벨 → 강아지 사진에 "고양이" 라벨 → AI가 혼란 문제 2: 편향된 데이터 → 페르시안 고양이만 9,000장 → 샴 고양이 100장 → AI가 페르시안만 잘 인식 문제 3: 저품질 이미지 → 흐릿한 사진, 너무 작은 사진 → AI가 패턴 못 찾음 → 10,000장 나쁜 데이터 < 1,000장 좋은 데이터

좋은 데이터 조건:

✅ 정확한 라벨 (고양이면 "고양이")

✅ 다양성 (여러 종류, 각도, 상황)

✅ 균형 (각 카테고리 비슷한 양)

✅ 깨끗함 (노이즈 없음)

Part 4. 데이터 편향 문제

편향이 뭐야?

데이터가 특정 쪽으로 치우쳐 있는 거야.

실제 사례: 얼굴 인식 AI (2018년 문제) → 백인 남성 데이터 90% → 흑인 여성 데이터 5% 결과: → 백인 남성: 정확도 99% → 흑인 여성: 정확도 65% → 논란, 재학습 필요 채용 AI (Amazon 사례) → 과거 합격자 데이터로 학습 → 과거 합격자는 남성이 많았음 결과: → "여성"이라는 단어만 있어도 감점 → 폐기 결정 → 데이터 편향 = AI 편향

교훈:

AI는 데이터 속 편견을 그대로 학습해.

데이터 수집할 때 다양성 확보가 필수!

Part 5. 데이터 없으면 정말 못 만들어?

데이터가 아예 없으면 AI 못 만드는 거야?

몇 가지 방법이 있긴 해:

데이터 부족 해결법: 1. 전이 학습 (Transfer Learning) → 다른 AI가 배운 지식 재활용 → 예: ImageNet으로 학습한 AI를 고양이 인식에 재사용 → 데이터 100배 적어도 됨 2. 데이터 증강 (Data Augmentation) → 기존 데이터를 변형해서 늘림 → 예: 고양이 사진 회전, 밝기 조정, 확대/축소 → 1,000장 → 10,000장 효과 3. 합성 데이터 (Synthetic Data) → 가짜 데이터 만들기 → 예: 3D 모델링으로 가상 고양이 생성 → 무한정 만들 수 있음 4. Few-Shot Learning → 적은 데이터로 학습하는 알고리즘 → 예: 3~5장만 봐도 학습 가능 → 최신 연구 분야

하지만:

진짜 데이터가 최고야.

위 방법들은 어디까지나 "차선책"이야.

Part 6. 데이터가 돈이 된 시대

그래서 요즘 데이터가 귀하다는 거야?

"데이터는 21세기의 석유"

실제로 돈 주고 사기도 해.

데이터의 가치: 구글, 페이스북이 무료인 이유? → 너의 데이터를 수집하기 위해 → 검색어, 클릭, 좋아요, 댓글 → 이걸로 광고 AI 학습 → 광고 정확도 ↑ → 수익 ↑ 자율주행 회사들의 경쟁: → 주행 데이터 확보 경쟁 → 테슬라: 수백만 대 차량에서 데이터 수집 → 경쟁사: 돈 주고 데이터 구매 OpenAI가 ChatGPT 무료로 푼 이유: → 사용자 대화 데이터 수집 → 피드백으로 모델 개선 → GPT-4, GPT-5 학습에 활용

결론:

좋은 데이터 = AI 회사의 경쟁력

알고리즘은 논문으로 공개되지만, 데이터는 비밀이야.

📌 핵심 정리

  • 연료 - 데이터는 AI의 연료, 없으면 학습 불가
  • 양 - 많을수록 성능 향상, 보통 수만~수백만 개 필요
  • 품질 - 양보다 품질이 더 중요, 정확하고 다양해야 함
  • 편향 - 데이터 편향 = AI 편향, 다양성 필수
  • 경쟁력 - 좋은 데이터 = AI 회사의 핵심 자산
