TOPIC 16

AI 개발자는 뭘 하는 사람이야?

Q. 모델을 만드는 사람? 쓰는 사람?

Part 1. AI 개발자 종류

AI 개발자는 뭘 해? GPT 같은 거 만들어?

"AI 개발자"는 여러 종류야.

AI 개발자 스펙트럼: 1. AI 연구자 (AI Researcher) → GPT, BERT 같은 새 알고리즘 개발 → 논문 작성 → 예: OpenAI, Google Brain 연구원 2. ML 엔지니어 (ML Engineer) → 기존 모델 활용해서 서비스 만들기 → ChatGPT API로 챗봇 개발 → 모델 학습, 최적화 3. AI 응용 개발자 (AI Application Developer) → AI 기능 추가한 앱 개발 → "이 버튼 누르면 AI 추천" → 일반 개발 + AI API 사용 4. 데이터 과학자 (Data Scientist) → 데이터 분석, 예측 모델 → "다음 달 매출 예측" → 통계 + 머신러닝 → 다 다른 일을 해

Part 2. 실제로 뭘 해?

구체적으로 어떤 일을 하는데?

각자 다른 업무:

역할 주요 업무 하루 일과
AI 연구자 새 알고리즘 개발, 논문 논문 읽기, 실험, 코딩, 토론
ML 엔지니어 모델 학습, 배포, 최적화 데이터 준비, 학습, 성능 개선
AI 앱 개발자 AI 기능 통합 API 연결, UI 개발, 테스트
데이터 과학자 데이터 분석, 인사이트 도출 데이터 분석, 시각화, 보고서
ML 엔지니어의 하루: 09:00 - 데이터 확인 → 어제 수집된 데이터 검증 → 라벨링 오류 수정 11:00 - 모델 학습 → 새 데이터로 재학습 → 하이퍼파라미터 조정 14:00 - 성능 평가 → 정확도 측정 (85% → 87% 향상!) → A/B 테스트 준비 16:00 - 배포 → 새 모델 서버에 올리기 → 모니터링 설정 18:00 - 회의 → 다음 주 개선 계획 논의

Part 3. 필요한 스킬

AI 개발자가 되려면 뭘 배워야 해?

역할마다 필요한 스킬이 달라.

AI 연구자: 필수: - 수학 (선형대수, 확률, 미적분) - 딥러닝 이론 - Python (PyTorch, TensorFlow) - 논문 읽기/쓰기 - 영어 (논문, 발표) 학위: - 박사 선호 (석사 최소) ML 엔지니어: 필수: - Python - 머신러닝 라이브러리 (scikit-learn, PyTorch) - 데이터 처리 (pandas, numpy) - 클라우드 (AWS, GCP) - Git 선택: - 수학 (기초만) - 학위 무관 (실력 중심) AI 앱 개발자: 필수: - 일반 프로그래밍 (Python, JavaScript) - API 사용법 - 웹/앱 개발 선택: - AI 이론 (깊이 몰라도 됨) - ChatGPT API 문서만 읽어도 충분 데이터 과학자: 필수: - 통계 - Python (pandas, matplotlib) - SQL - 비즈니스 이해 선택: - 딥러닝 (간단한 모델만)

Part 4. 나는 어디로?

나는 어떤 길로 가야 해?

목표에 따라 다른 경로

경로별 가이드: GPT 같은 모델 만들고 싶다: → AI 연구자 → 필요: 박사 학위, 수학, 논문 → 난이도: ★★★★★ → 채용: Google, OpenAI, Meta (극소수) AI 서비스 만들고 싶다: → ML 엔지니어 → 필요: 코딩, 머신러닝 기초 → 난이도: ★★★★☆ → 채용: 스타트업, 중견기업 (많음) AI 기능 앱 만들고 싶다: → AI 앱 개발자 → 필요: 웹/앱 개발, API 사용 → 난이도: ★★★☆☆ → 채용: 모든 IT 회사 (매우 많음) 데이터로 인사이트 찾고 싶다: → 데이터 과학자 → 필요: 통계, 비즈니스, SQL → 난이도: ★★★☆☆ → 채용: 금융, 커머스, 제조 (많음) 초보자 추천: → AI 앱 개발자부터 시작 → ChatGPT API 활용한 간단한 프로젝트 → 점차 ML 엔지니어로 성장

Part 5. 실제 취업 시장

AI 개발자 취업 잘 돼?

2024년 기준, 매우 좋아.

취업 시장 현황: 수요: - AI 관련 채용 공고 급증 - ChatGPT 이후 폭발적 증가 - 모든 회사가 "AI 도입" 원함 연봉: - 신입 ML 엔지니어: 5000만~7000만 - 경력 3년: 8000만~1억 - AI 연구자 (박사): 1억~2억+ - (2024년 한국 기준) 미국: - 신입: $120k~$180k - 경력: $200k~$500k+ 필요 역량: 포트폴리오 필수: - GitHub에 프로젝트 - Kaggle 대회 참여 - 개인 블로그 운영 실무 경험: - 인턴, 부트캠프 - 오픈소스 기여 - 해커톤 진입 장벽: - 예전: 매우 높음 (박사 필수) - 지금: 낮아짐 (독학 가능) - 하지만 경쟁 치열

Part 6. 미래 전망

AI가 AI 개발자를 대체하지 않을까?

역설적으로, AI가 발전할수록 AI 개발자 수요 증가.

미래 전망: AI가 못 하는 것: 1. 문제 정의 → "무엇을 만들지" 결정 → 비즈니스 이해 필요 2. 데이터 준비 → "어떤 데이터를 수집할지" → 도메인 지식 필요 3. 윤리적 판단 → "편향 제거", "공정성" → 사람의 가치 판단 필요 4. 창의적 문제 해결 → 새로운 접근법 → AI는 패턴만 따름 변화하는 역할: 과거 (2020년 이전): → 코딩 많이 함 → 알고리즘 직접 구현 현재 (2024년): → 라이브러리 활용 → 모델 선택, 조합 미래 (2030년?): → AI가 코딩 대신 → 사람은 전략, 판단 → "AI 매니저" 역할 → 일의 성격은 바뀌지만 → 수요는 오히려 증가

핵심:

AI 개발자 = AI를 만드는 사람 + AI를 관리하는 사람

앞으로는 후자가 더 중요해질 거야.

📌 핵심 정리

  • 다양한 역할 - 연구자, ML 엔지니어, 앱 개발자, 데이터 과학자
  • 필요 스킬 - 역할마다 다름, 수학부터 API 사용까지
  • 진입 방법 - AI 앱 개발자부터 시작 추천

  • 미래 - AI가 발전해도 AI 개발자 수요는 증가
  • 역할 변화 - 코딩 → 전략/판단으로 변화
