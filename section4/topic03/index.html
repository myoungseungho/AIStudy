TOPIC 03

가중치와 편향은 뭐야?

Q. 학습한다는 게 이 값들을 조정하는 거야?

Part 1. 가중치(Weight)가 뭐야?

가중치는 뭐야?

각 입력의 중요도야.

생각해봐. 면접 점수를 계산할 때 "학력 30%, 경력 50%, 면접 20%" 이런 식으로 비율을 정하잖아?

이 비율이 바로 가중치야.

예시: 대출 승인 AI 입력: - 연봉 - 신용점수 - 재직기간 가중치가 다르면 결과도 달라져: 케이스 1: 연봉을 중시 출력 = 0.7×연봉 + 0.2×신용점수 + 0.1×재직기간 → 연봉이 높으면 쉽게 승인 케이스 2: 신용점수를 중시 출력 = 0.2×연봉 + 0.7×신용점수 + 0.1×재직기간 → 신용이 좋으면 쉽게 승인 케이스 3: 균형잡힌 판단 출력 = 0.4×연봉 + 0.4×신용점수 + 0.2×재직기간 → 모든 요소를 골고루 고려

이해됐어?

가중치 = 어떤 입력을 얼마나 중요하게 볼지 결정하는 숫자

Part 2. 편향(Bias)은 뭐야?

편향은 또 뭐야?

기준점을 조정하는 값이야.

시험 합격선을 생각해봐.

편향의 역할: 편향 없이: 점수 = 0.5×필기 + 0.3×면접 + 0.2×출석 합격 기준: 점수 > 0 → 모든 과목이 0점이어도 합격? 이상하지? 편향 추가: 점수 = 0.5×필기 + 0.3×면접 + 0.2×출석 - 40 합격 기준: 점수 > 0 → 40점 이상의 가중합이 필요 → 최소 기준이 생김! 편향을 바꾸면: b = -30 → 더 쉽게 합격 (합격선 낮춤) b = -50 → 더 어렵게 합격 (합격선 높임) b = 0 → 기준 없음 (비추천)

편향 = 합격선, 기준점, 임계값

입력과 무관하게 항상 더해지는 상수값이야.

Part 3. 가중치 vs 편향, 뭐가 달라?

구분 가중치 (Weight) 편향 (Bias)
역할 입력의 중요도 기준점 조정
개수 입력 개수만큼 퍼셉트론당 1개
계산 입력과 곱셈 그냥 더함
영향 입력에 비례 입력과 무관
비유 볼륨 조절 기본 음량
그림으로 이해: 직선의 방정식: y = wx + b w (가중치): - 직선의 기울기 - 입력 변화에 얼마나 민감한지 b (편향): - 직선의 y절편 - 기본적으로 얼마나 활성화될지 w=2, b=0: y = 2x (원점 통과, 가파른 기울기) w=1, b=3: y = x + 3 (위로 이동, 완만한 기울기) w=0.5, b=-2: y = 0.5x - 2 (아래로 이동, 더 완만)

가중치 = 방향과 강도

편향 = 기본 설정값

Part 4. "학습"은 이 값들을 조정하는 거야?

맞아! 학습한다는 게 이 값들을 바꾸는 거야?

정확히 맞았어!

신경망 학습 = 가중치와 편향을 최적화하는 과정

학습 과정: 1. 초기화 가중치 = 랜덤한 작은 값 편향 = 0 또는 작은 값 2. 예측 출력 = f(w₁x₁ + w₂x₂ + ... + b) 3. 오차 계산 오차 = |실제값 - 예측값| 4. 가중치 조정 w_new = w_old - 학습률 × (오차에 대한 w의 기여도) 5. 편향 조정 b_new = b_old - 학습률 × (오차에 대한 b의 기여도) 6. 반복 오차가 충분히 작아질 때까지 2~5 반복 예시: 초기: w=0.1, b=0.5 예측: 3.5 정답: 5.0 오차: 1.5 조정 후: w=0.15, b=0.7 예측: 4.2 정답: 5.0 오차: 0.8 조정 후: w=0.18, b=0.85 예측: 4.7 정답: 5.0 오차: 0.3 ... (계속 조정)

이해됐지?

AI는 코드로 짜여진 게 아니야.

데이터를 보면서 가중치를 조정한 거야.

Part 5. 가중치만 있으면 안 돼?

편향이 왜 필요한데? 가중치만으로 충분하지 않아?

가중치만 있으면 표현력이 제한돼.

편향이 없으면: y = w₁x₁ + w₂x₂ 문제점: 1. 항상 원점(0,0)을 지나감 2. 결정 경계가 제한적 3. 데이터가 원점 근처에 없으면 학습 어려움 편향이 있으면: y = w₁x₁ + w₂x₂ + b 장점: 1. 원점과 무관하게 이동 가능 2. 결정 경계를 자유롭게 배치 3. 다양한 데이터 분포에 대응 실제 예시: 편향 없음: - "키 × 0.5 + 몸무게 × 0.3" - 키와 몸무게가 모두 0이면 무조건 0 - 작은 사람은 항상 불리 편향 있음: - "키 × 0.5 + 몸무게 × 0.3 - 50" - 최소 기준선 설정 가능 - 공정한 판단 가능

편향은 모델에 유연성을 준다고 생각하면 돼.

Part 6. 실제 신경망의 가중치는 어떻게 생겼어?

실제로 어떤 값들이야?

코드로 보여줄게.

import numpy as np # 간단한 신경망 만들기 class SimpleNN: def __init__(self): # 가중치 초기화 (2개 입력, 3개 뉴런) self.W = np.random.randn(2, 3) * 0.01 # 편향 초기화 self.b = np.zeros(3) def forward(self, x): return np.dot(x, self.W) + self.b # 신경망 생성 nn = SimpleNN() print("가중치:") print(nn.W) # 출력 예: # [[ 0.0043 -0.0091 0.0052] # [-0.0071 0.0034 0.0089]] print("\n편향:") print(nn.b) # 출력: [0. 0. 0.] # 예측 x = np.array([1.5, 2.3]) output = nn.forward(x) print("\n출력:") print(output) # 출력 예: [-0.0098 0.0014 0.0282] # 학습 후에는 이렇게 바뀜: # 가중치: # [[ 2.3451 -1.2341 0.8923] # [-0.5612 1.7834 2.1234]] # 편향: # [-0.5 0.3 1.2]

보이지?

그냥 숫자들의 배열이야.

ChatGPT 같은 거대 모델도 결국 수십억 개의 이런 숫자들일 뿐이야.

핵심 정리

  • 가중치 = 입력의 중요도 - 각 입력에 곱해지는 값
  • 편향 = 기준점 조정 - 항상 더해지는 상수값
  • 학습 = 가중치와 편향 조정 - 오차를 줄이는 방향으로 값 변경
  • 둘 다 필요 - 가중치는 방향, 편향은 위치를 조절
  • 실체는 숫자 - 거대한 모델도 결국 숫자 배열
